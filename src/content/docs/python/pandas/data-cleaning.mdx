For time series with a `DatetimeIndex`, use `method="time"` to interpolate based on actual time gaps rather than assuming equal spacing. This is critical for unevenly sampled sensor data or financial tick data.

## Outlier Detection

```python
data = pd.Series([22, 25, 21, 24, 28, 200, 23, 26, 19, 24, -50, 27])

# IQR method
Q1 = data.quantile(0.25)
Q3 = data.quantile(0.75)
IQR = Q3 - Q1

lower_bound = Q1 - 1.5 * IQR
upper_bound = Q3 + 1.5 * IQR

outliers = data[(data < lower_bound) | (data > upper_bound)]
clean    = data[(data >= lower_bound) & (data <= upper_bound)]

print(f"Q1={Q1}, Q3={Q3}, IQR={IQR}")
print(f"Bounds: [{lower_bound:.1f}, {upper_bound:.1f}]")
print(f"Outliers: {outliers.tolist()}")
print(f"Clean data: {clean.tolist()}")
```

```text
Q1=22.25, Q3=26.25, IQR=4.0
Bounds: [16.2, 32.2]
Outliers: [200, -50]
Clean data: [22, 25, 21, 24, 28, 23, 26, 19, 24, 27]
```

```python
# Cap outliers (Winsorization) instead of dropping
capped = data.clip(lower=lower_bound, upper=upper_bound)
print(f"Capped: {capped.tolist()}")
```

```text
Capped: [22, 25, 21, 24, 28, 32.2, 23, 26, 19, 24, 16.2, 27]
```

:::caution[Choose Your Outlier Strategy Carefully]
Removing outliers can introduce bias if they are legitimate extreme values (e.g., a billionaire's income in a salary dataset). Always investigate why a value is extreme before deciding to drop, cap, or transform it. Document every decision.
:::

## where() and mask()

`where()` keeps values where the condition is `True` and replaces others. `mask()` is the inverse.

```python
scores = pd.Series([4.5, 2.1, 3.8, 1.5, 4.9, 3.2])

# where(): keep value if condition True, else replace with NaN (or given value)
# Keep scores above 3.0; replace the rest with NaN
print(scores.where(scores >= 3.0))
```

```text
0    4.5
1    NaN
2    3.8
3    NaN
4    4.9
5    3.2
dtype: float64
```

```python
# mask(): replace where condition is True (inverse of where)
# Replace scores below 3.0 with 0 (failed threshold)
print(scores.mask(scores < 3.0, other=0.0))
```

```text
0    4.5
1    0.0
2    3.8
3    0.0
4    4.9
5    3.2
dtype: float64
```

```python
# Apply to a DataFrame column — cap values at a threshold
df_scores = pd.DataFrame({"score": [88, 102, 95, 110, 78]})
df_scores["capped"] = df_scores["score"].where(df_scores["score"] <= 100, other=100)
```

## Complete Cleaning Pipeline

```python
def clean_dataset(raw: pd.DataFrame) -> pd.DataFrame:
    df = raw.copy()

    # 1. Remove exact duplicate rows
    df = df.drop_duplicates()

    # 2. Strip whitespace and normalize case
    df["name"]       = df["name"].str.strip().str.title()
    df["email"]      = df["email"].str.strip().str.lower()
    df["department"] = df["department"].str.strip().str.title()

    # 3. Fix known value inconsistencies
    df["department"] = df["department"].replace({"Hr": "HR"})

    # 4. Convert salary to numeric
    df["salary"] = pd.to_numeric(
        df["salary"].astype(str).str.replace(",", "", regex=False),
        errors="coerce",
    )

    # 5. Parse all date formats
    df["hire_date"] = pd.to_datetime(df["hire_date"], dayfirst=False,
                                      infer_datetime_format=True, errors="coerce")

    # 6. Fill missing numeric values with median
    df["salary"] = df["salary"].fillna(df["salary"].median())
    df["score"]  = df["score"].fillna(df["score"].median())

    # 7. Fill missing strings with placeholder
    df["name"]  = df["name"].fillna("Unknown")
    df["email"] = df["email"].fillna("no-email@unknown.com")

    # 8. Remove remaining rows still missing critical fields
    df = df.dropna(subset=["hire_date"])

    return df.reset_index(drop=True)

clean = clean_dataset(raw)
print(clean)
```

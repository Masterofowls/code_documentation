This guide takes you from zero to your first working data exploration in Pandas. By the end you will know how to install Pandas, create Series and DataFrames from various sources, and run a complete exploration workflow on any new dataset.

## Installation

<Steps>

1. **Install via pip** (recommended for most environments)

   ```bash
   pip install pandas
   ```

   For a full data science stack:

   ```bash
   pip install "pandas[performance]"   # adds bottleneck + numexpr
   pip install pandas numpy matplotlib seaborn scikit-learn
   ```

2. **Install via conda / mamba** (Anaconda / Miniconda environments)

   ```bash
   conda install -c conda-forge pandas
   # or with mamba (faster solver)
   mamba install -c conda-forge pandas
   ```

3. **Install via uv** (modern Python tooling)

   ```bash
   uv add pandas
   # or within a project
   uv pip install pandas
   ```

4. **Verify the installation**

   ```python
   import pandas as pd
   print(pd.__version__)
   ```

   ```text
   2.2.2
   ```

</Steps>

:::tip[Recommended Version]
Use **Pandas 2.x** for all new projects. Pandas 2.0 introduced Copy-on-Write semantics (prevents accidental DataFrame mutations), optional Apache Arrow-backed dtypes, and removed the deprecated `DataFrame.append()`. If upgrading from 1.x, check downstream library compatibility first.
:::

## Import Conventions

The universally accepted alias for Pandas is `pd`. You will almost always also import NumPy.

```python
import pandas as pd
import numpy as np

# Common companions
import matplotlib.pyplot as plt
import seaborn as sns

# Confirm versions
print(f"pandas  {pd.__version__}")
print(f"numpy   {np.__version__}")
```

```text
pandas  2.2.2
numpy   1.26.4
```

:::caution[Avoid Wildcard Imports]
Never use `from pandas import *`. It pollutes your namespace and makes it impossible to tell which functions come from Pandas vs NumPy vs standard Python. Always prefix with `pd.` — it is a critical readability signal in any shared codebase.
:::

## Creating a Series

A Series is a one-dimensional labeled array. It is the building block of DataFrames.

<Tabs>
  <TabItem label="From a List">

```python
import pandas as pd

# Auto-generated integer index (0, 1, 2, ...)
temperatures = pd.Series([22.4, 19.1, 25.8, 23.0, 18.5])
print(temperatures)
```

```text
0    22.4
1    19.1
2    25.8
3    23.0
4    18.5
dtype: float64
```

```python
# Provide a custom index and name
temperatures = pd.Series(
    [22.4, 19.1, 25.8, 23.0, 18.5],
    index=["Mon", "Tue", "Wed", "Thu", "Fri"],
    name="temp_celsius",
)
print(temperatures)
print(f"\nMax: {temperatures.max()} on {temperatures.idxmax()}")
print(f"Min: {temperatures.min()} on {temperatures.idxmin()}")
print(f"Avg: {temperatures.mean():.2f}")
```

### info()

`info()` displays the schema — column names, non-null counts, dtypes, and memory usage. It is the fastest way to get an overview of a new dataset.



:::tip[Spot Missing Values Instantly]
When `info()` shows a non-null count less than the total row count for any column, that column contains missing values. This is the fastest way to detect `NaN`s before doing deeper analysis.

:::caution[Dot Notation Limitations]
Avoid `df.colname` in production code. It fails when the column name conflicts with a DataFrame attribute or method (e.g., `df.count`, `df.mean`, `df.index`), and it does not work for names containing spaces or special characters. Always use `df["colname"]` in scripts and notebooks intended for sharing.
:::

## Missing Value Overview

```python
# Create a DataFrame with missing values
import numpy as np

messy = pd.DataFrame({
    "id":     [1, 2, 3, 4, 5],
    "name":   ["Alice", None, "Carol", "David", None],
    "score":  [88.0, 92.0, np.nan, 79.0, 85.0],
    "grade":  ["A", "A", None, "B", "B"],
})

print(messy)
print()
print("Missing values per column:")
print(messy.isnull().sum())
print()
print("Missing value percentage:")
print((messy.isnull().sum() / len(messy) * 100).round(1))
```

## Quick Exploration Workflow

This is a reusable pattern for the first look at any new dataset:

```python
import pandas as pd

def explore(df: pd.DataFrame, name: str = "DataFrame") -> None:
    """Run a complete first-look exploration on any DataFrame."""
    print(f"{'='*60}")
    print(f"  {name}")
    print(f"{'='*60}")

    # 1. Dimensions
    print(f"\n[1] Shape: {df.shape[0]:,} rows × {df.shape[1]} columns")

    # 2. Schema
    print("\n[2] Schema (info):")
    df.info(memory_usage="deep")

    # 3. Numeric summary
    print("\n[3] Numeric summary:")
    print(df.describe().round(2))

    # 4. Missing values
    missing = df.isnull().sum()
    missing = missing[missing > 0]
    if len(missing):
        print("\n[4] Missing values:")
        pct = (missing / len(df) * 100).round(2)
        print(pd.concat([missing, pct], axis=1, keys=["count", "pct%"]))
    else:
        print("\n[4] Missing values: None")

    # 5. Unique counts
    print("\n[5] Unique values per column:")
    print(df.nunique().sort_values())

    # 6. Sample rows
    print("\n[6] Random sample:")
    print(df.sample(min(3, len(df)), random_state=42))

# Usage
df = pd.read_csv("employees.csv")
explore(df, "Employees")
```

## Reference: Core Properties and Methods

| Property / Method | Returns | Description |
|---|---|---|
| `df.shape` | `(int, int)` | `(n_rows, n_cols)` tuple |
| `df.dtypes` | Series | dtype for each column |
| `df.columns` | Index | Column names |
| `df.index` | Index | Row labels |
| `df.size` | int | Total elements (rows × cols) |
| `df.ndim` | int | Always 2 for DataFrames |
| `df.memory_usage(deep=True)` | Series | Bytes per column |
| `df.head(n)` | DataFrame | First n rows |
| `df.tail(n)` | DataFrame | Last n rows |
| `df.sample(n)` | DataFrame | Random n rows |
| `df.info()` | None | Prints schema to stdout |
| `df.describe()` | DataFrame | Summary statistics |
| `df.isnull().sum()` | Series | Missing count per column |
| `df.notnull().sum()` | Series | Non-missing count per column |
| `df.nunique()` | Series | Distinct values per column |
| `df["col"].value_counts()` | Series | Frequency of each value |
| `df["col"].unique()` | ndarray | Array of unique values |

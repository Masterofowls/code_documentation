Pandas provides a high-level `.plot()` API that delegates to Matplotlib under the hood. This lets you create charts directly from DataFrames and Series with minimal code, while retaining full access to Matplotlib's customization when needed.

## Setup

```python
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
import matplotlib.ticker as mticker
import seaborn as sns

# Consistent styling for all examples
plt.rcParams.update({
    "figure.figsize":   (10, 5),
    "figure.dpi":       100,
    "axes.spines.top":  False,
    "axes.spines.right":False,
    "axes.grid":        True,
    "grid.alpha":       0.3,
    "font.size":        11,
})

# Seaborn theme (optional — also affects Matplotlib plots)
sns.set_theme(style="whitegrid", palette="tab10")
```

## The df.plot() API

Every `DataFrame.plot()` call returns a Matplotlib `Axes` object, giving you full control.

```python
# Syntax overview
ax = df.plot(
    kind="line",           # chart type: line, bar, barh, hist, box, scatter,
                           #             pie, area, hexbin, kde, density
    x=None,                # column for x-axis (index used if None)
    y=None,                # column(s) for y-axis (all numeric if None)
    figsize=(10, 5),       # (width, height) in inches
    title="My Chart",      # axes title
    xlabel="X Label",      # x-axis label
    ylabel="Y Label",      # y-axis label
    color=None,            # color or list of colors
    style=None,            # Matplotlib linestyle per column (e.g., "--")
    alpha=1.0,             # transparency 0.0–1.0
    legend=True,           # show legend
    grid=True,             # show grid
    rot=0,                 # x-tick label rotation
    fontsize=None,         # font size for tick labels
    logx=False,            # log scale on x-axis
    logy=False,            # log scale on y-axis
    xlim=None,             # (min, max) for x-axis
    ylim=None,             # (min, max) for y-axis
    subplots=False,        # each column in its own subplot
    layout=None,           # (rows, cols) for subplot grid
    sharex=False,          # share x-axis across subplots
    sharey=False,          # share y-axis across subplots
    ax=None,               # existing Axes to draw on
)
```

## Line Chart

Line charts are ideal for time series and trends over an ordered axis.

```python
ax = monthly.plot(
    kind="line",
    figsize=(12, 5),
    title="Monthly Revenue by Region — 2024",
    xlabel="Month",
    ylabel="Revenue ($)",
    marker="o",         # mark data points
    linewidth=2,
    alpha=0.85,
)

ax.yaxis.set_major_formatter(mticker.FuncFormatter(lambda x, _: f"${x/1000:.0f}K"))
ax.legend(title="Region", loc="upper left")
plt.tight_layout()
plt.savefig("line_chart.png", dpi=150, bbox_inches="tight")
plt.show()
```

**Expected output:** A line chart with three colored lines (North, South, East) showing upward revenue trends from January through June. North has the highest revenue, East the lowest. Data points are marked with circles.

## Bar Chart (Vertical)

Bar charts compare categorical values.

```python
ax = products.set_index("product")["revenue"].plot(
    kind="bar",
    figsize=(10, 5),
    title="Revenue by Product",
    color=["#2196F3", "#4CAF50", "#FF9800", "#F44336", "#9C27B0"],
    edgecolor="white",
    width=0.65,
    rot=0,
)
ax.yaxis.set_major_formatter(mticker.FuncFormatter(lambda x, _: f"${x/1000:.0f}K"))
ax.set_xlabel("")
ax.set_ylabel("Total Revenue")

# Add value labels on top of each bar
for bar in ax.patches:
    ax.text(
        bar.get_x() + bar.get_width() / 2,
        bar.get_height() + 300,
        f"${bar.get_height()/1000:.1f}K",
        ha="center", va="bottom", fontsize=9,
    )

plt.tight_layout()
plt.show()
```

**Expected output:** Five vertical bars, one per product, with Laptop being the tallest and Mouse the shortest. Each bar has a dollar value label at the top.

## Box Plot

Box plots show median, quartiles, and outliers.

```python
ax = scores.plot(
    kind="box",
    figsize=(8, 6),
    title="Score Distribution — Box Plot",
    patch_artist=True,        # fill boxes with color
    notch=False,
    vert=True,
)
ax.set_ylabel("Score")
plt.tight_layout()
plt.show()
```

**Expected output:** Three box plots side by side showing the IQR, median line, whiskers extending to 1.5×IQR, and individual outlier points. Science has the widest spread (highest std).

## Scatter Plot

Scatter plots reveal relationships between two numeric variables.

```python
ax = products.plot(
    kind="scatter",
    x="avg_price",
    y="revenue",
    figsize=(9, 6),
    title="Revenue vs Average Price",
    s=products["units_sold"] / 5,   # bubble size = units sold
    c="return_rate",                 # color = return rate
    colormap="RdYlGn_r",
    alpha=0.75,
    edgecolors="black",
    linewidths=0.5,
)
ax.set_xlabel("Average Unit Price ($)")
ax.set_ylabel("Total Revenue ($)")

# Annotate each point
for _, row in products.iterrows():
    ax.annotate(
        row["product"],
        xy=(row["avg_price"], row["revenue"]),
        xytext=(5, 5),
        textcoords="offset points",
        fontsize=9,
    )
plt.tight_layout()
plt.show()
```

**Expected output:** Five bubbles of varying size (proportional to units sold). Laptop appears as a large bubble at high price / high revenue. Mouse is a large bubble at low price / low revenue. Color gradient indicates return rate.

## Pie Chart

```python
ax = products.set_index("product")["revenue"].plot(
    kind="pie",
    figsize=(9, 9),
    title="Revenue Share by Product",
    autopct="%1.1f%%",
    startangle=90,
    pctdistance=0.75,
    labeldistance=1.1,
    wedgeprops={"edgecolor": "white", "linewidth": 1.5},
    explode=[0.05, 0, 0, 0, 0],   # slightly explode the first slice
)
ax.set_ylabel("")
plt.tight_layout()
plt.show()
```

**Expected output:** A pie chart with five slices. Laptop dominates with ~46.5% of revenue. Mouse is the smallest slice at ~5%.

:::caution[When Not to Use Pie Charts]
Pie charts are poor for comparing more than five categories or when values are similar. Use bar charts instead — humans judge height (linear) far more accurately than angle or arc length. Reserve pie charts for simple "part of whole" narratives.
:::

## Hexbin Plot

Hexbin plots are scatter plot alternatives for large datasets where overplotting is an issue.

```python
large = pd.DataFrame({
    "x": np.random.normal(0, 1, 5000),
    "y": np.random.normal(0, 1, 5000) + 0.5 * np.random.normal(0, 1, 5000),
})

ax = large.plot(
    kind="hexbin",
    x="x",
    y="y",
    gridsize=25,
    colormap="YlOrRd",
    figsize=(9, 7),
    title="Hexbin Density Plot (n=5,000)",
)
ax.set_xlabel("X Value")
ax.set_ylabel("Y Value")
plt.tight_layout()
plt.show()
```

**Expected output:** A hexagonal grid where each hexagon's color represents the count of data points in that bin. The densest hexagons (darkest color) are near the origin where both distributions peak.

## Matplotlib Integration

When you need fine-grained control, create `Figure` and `Axes` objects first and pass them to `.plot()`.

```python
fig, axes = plt.subplots(1, 2, figsize=(14, 5))
fig.suptitle("Revenue Analysis Dashboard", fontsize=15, fontweight="bold")

# Left plot: line chart
monthly.plot(ax=axes[0], title="Monthly Trend", marker="o", linewidth=2)
axes[0].set_ylabel("Revenue ($)")
axes[0].yaxis.set_major_formatter(mticker.FuncFormatter(lambda x, _: f"${x/1000:.0f}K"))

# Right plot: bar chart
products.set_index("product")["revenue"].sort_values(ascending=True).plot(
    kind="barh",
    ax=axes[1],
    title="Revenue by Product",
    color="#42A5F5",
    edgecolor="white",
)
axes[1].xaxis.set_major_formatter(mticker.FuncFormatter(lambda x, _: f"${x/1000:.0f}K"))
axes[1].set_xlabel("Revenue")

plt.tight_layout()
plt.savefig("dashboard.png", dpi=150, bbox_inches="tight")
plt.show()
```

**Expected output:** A two-panel figure. Left panel shows three trend lines for regional revenue. Right panel shows a horizontal bar chart of products sorted by revenue. Both share the same figure title.

## Time Series Visualization

```python
# Generate a time series with trend + seasonality + noise
dates = pd.date_range("2023-01-01", periods=365, freq="D")
trend = np.linspace(1000, 1500, 365)
seasonal = 200 * np.sin(2 * np.pi * np.arange(365) / 365)
noise = np.random.normal(0, 50, 365)

ts = pd.Series(trend + seasonal + noise, index=dates, name="daily_revenue")

# Resample to weekly and monthly for aggregated views
weekly  = ts.resample("W").mean()
monthly_ts = ts.resample("ME").sum()

fig, axes = plt.subplots(3, 1, figsize=(14, 12), sharex=False)

# Panel 1: Raw daily data
ts.plot(ax=axes[0], alpha=0.5, color="#90CAF9", linewidth=0.8)
ts.rolling(30).mean().plot(ax=axes[0], color="#1565C0", linewidth=2, label="30-day MA")
axes[0].set_title("Daily Revenue (raw + 30-day moving average)")
axes[0].legend()

# Panel 2: Weekly aggregation
weekly.plot(ax=axes[1], kind="area", alpha=0.4, color="#A5D6A7")
weekly.plot(ax=axes[1], color="#2E7D32", linewidth=1.5)
axes[1].set_title("Weekly Average Revenue")

# Panel 3: Monthly totals as bars
monthly_ts.plot(ax=axes[2], kind="bar", color="#FFF176", edgecolor="#F57F17",
                rot=45)
axes[2].set_title("Monthly Total Revenue")
axes[2].yaxis.set_major_formatter(mticker.FuncFormatter(lambda x, _: f"${x/1000:.0f}K"))

plt.tight_layout()
plt.savefig("timeseries.png", dpi=150, bbox_inches="tight")
plt.show()
```

**Expected output:** Three stacked panels. Top panel shows jagged daily revenue with a smooth 30-day moving average overlay. Middle panel shows smoother weekly averages as a filled area. Bottom panel shows monthly totals as vertical bars with visible seasonality (revenues peak mid-year).

## Saving Figures

```python
# PNG — best for web and general use
plt.savefig("chart.png", dpi=150, bbox_inches="tight", transparent=False)

# SVG — scalable vector, best for publications and web
plt.savefig("chart.svg", bbox_inches="tight")

# PDF — for print-quality reports
plt.savefig("chart.pdf", bbox_inches="tight")

# High-resolution PNG for print
plt.savefig("chart_hires.png", dpi=300, bbox_inches="tight")

# Save with a white background (overrides transparent figure)
fig.patch.set_facecolor("white")
plt.savefig("chart_white.png", dpi=150, bbox_inches="tight", facecolor="white")
```

:::tip[Use bbox_inches="tight"]
Always pass `bbox_inches="tight"` to `savefig()`. Without it, long axis labels and legends outside the axes boundaries will be clipped in the saved image. This is the most common reason saved figures look different from the displayed figure in Jupyter.
:::

## Plot Parameters Quick Reference

| Parameter           | Type         | Description                                                                                                     |
| ------------------- | ------------ | --------------------------------------------------------------------------------------------------------------- |
| `kind`              | str          | Chart type: `"line"`, `"bar"`, `"barh"`, `"hist"`, `"box"`, `"scatter"`, `"pie"`, `"area"`, `"hexbin"`, `"kde"` |
| `figsize`           | tuple        | Figure size `(width, height)` in inches                                                                         |
| `title`             | str / list   | Axes title (list for subplots)                                                                                  |
| `xlabel` / `ylabel` | str          | Axis labels                                                                                                     |
| `color`             | str / list   | Single color or list per series                                                                                 |
| `colormap`          | str          | Matplotlib colormap name                                                                                        |
| `alpha`             | float        | Transparency 0.0–1.0                                                                                            |
| `style`             | str / list   | Matplotlib format string (e.g., `"--o"`)                                                                        |
| `marker`            | str          | Marker style for line charts                                                                                    |
| `linewidth`         | float        | Line width in points                                                                                            |
| `legend`            | bool         | Show or hide legend                                                                                             |
| `grid`              | bool         | Show gridlines                                                                                                  |
| `rot`               | int          | x-tick label rotation in degrees                                                                                |
| `xlim` / `ylim`     | tuple        | Axis limits `(min, max)`                                                                                        |
| `logx` / `logy`     | bool         | Logarithmic scale                                                                                               |
| `subplots`          | bool         | One subplot per column                                                                                          |
| `layout`            | tuple        | `(rows, cols)` for subplot grid                                                                                 |
| `sharex` / `sharey` | bool         | Shared axes across subplots                                                                                     |
| `ax`                | Axes         | Existing Matplotlib Axes to draw on                                                                             |
| `stacked`           | bool         | Stacked bars or areas                                                                                           |
| `bins`              | int          | Histogram bin count                                                                                             |
| `edgecolor`         | str          | Bar / histogram edge color                                                                                      |
| `width`             | float        | Bar width (0–1)                                                                                                 |
| `gridsize`          | int          | Hexbin grid resolution                                                                                          |
| `s`                 | int / Series | Scatter point size                                                                                              |
| `c`                 | str / Series | Scatter point color (column name or values)                                                                     |

:::tip[Always Assign the Axes Object]

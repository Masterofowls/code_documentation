The game loop is the heartbeat of every Pygame application. Getting it right means consistent gameplay speed across all hardware, clean state management, and a codebase that can grow without becoming spaghetti.

## Fixed Timestep vs Variable Timestep

<Tabs>
  <TabItem label="Variable (delta time)">
```python
clock = pygame.time.Clock()

while running:
    dt = clock.tick(60) / 1000.0  # Seconds since last frame

    # Multiply all movement/physics by dt
    player.x += player.vx * dt * 300   # 300 pixels/second
    player.y += player.vy * dt * 300

    # Drawback: can accumulate floating-point error over time
    # Advantage: naturally handles frame-rate drops
```
  </TabItem>
  <TabItem label="Fixed timestep (recommended)">
```python
FIXED_DT      = 1 / 60        # 60 physics ticks per second
MAX_FRAME_TIME = 0.25          # Prevent spiral of death

clock         = pygame.time.Clock()
accumulator   = 0.0

while running:
    frame_time  = min(clock.tick() / 1000.0, MAX_FRAME_TIME)
    accumulator += frame_time

    # Process events once per render frame
    for event in pygame.event.get():
        handle_event(event)

    # Run physics in fixed steps
    while accumulator >= FIXED_DT:
        update_physics(FIXED_DT)
        accumulator -= FIXED_DT

    # Interpolate rendering position
    alpha  = accumulator / FIXED_DT   # 0.0 – 1.0
    render(alpha)

    pygame.display.flip()
```
  </TabItem>
  <TabItem label="Simple (no delta)">
```python
clock = pygame.time.Clock()

# Cap at 60 FPS; movement is in pixels/frame (not pixels/second)
# Simple and fine for small games with stable frame rate

while running:
    player.x += player.vx   # pixels per frame
    player.y += player.vy

    pygame.display.flip()
    clock.tick(60)           # Sleep until 1/60s has elapsed
```
  </TabItem>
</Tabs>

:::tip[Which to Choose?]
- **Small / jam games**: simple no-delta loop is fine at locked 60 FPS.
- **Multiplayer or physics**: fixed timestep with interpolated rendering is the gold standard.
- **General single-player**: variable delta time with `clock.tick(60)` is the sweet spot.
:::

## `Clock.tick()` vs `tick_busy_loop()`

```python
clock = pygame.time.Clock()

# tick() — uses OS sleep (yields CPU, saves battery, may overshoot by a frame)
dt = clock.tick(60)

# tick_busy_loop() — busy-waits (very precise timing, uses 100% CPU)
dt = clock.tick_busy_loop(60)

# Get actual measured frame rate (moving average)
fps = clock.get_fps()

# Get time since last tick without capping (use in fixed-timestep accumulator)
raw_dt = clock.tick()
```

```python
font = pygame.font.SysFont("monospace", 16)

# In the draw section:
fps_surf = font.render(f"FPS: {clock.get_fps():.0f}", True, (200, 255, 200))

```

## State Machine Pattern

Games naturally transition between distinct states. A simple enum + dispatch pattern keeps the loop clean:

```python
from enum import Enum, auto

class GameState(Enum):
    MENU      = auto()
    PLAYING   = auto()
    PAUSED    = auto()
    GAME_OVER = auto()
    WIN       = auto()
```

```python
state = GameState.MENU

while running:
    dt = clock.tick(60) / 1000.0

    for event in pygame.event.get():
        if event.type == pygame.QUIT:
            running = False

        if state == GameState.MENU:
            if event.type == pygame.KEYDOWN and event.key == pygame.K_RETURN:
                state = GameState.PLAYING

        elif state == GameState.PLAYING:
            if event.type == pygame.KEYDOWN:
                if event.key == pygame.K_ESCAPE:
                    state = GameState.PAUSED
                if event.key == pygame.K_SPACE:
                    player.jump()

        elif state == GameState.PAUSED:
            if event.type == pygame.KEYDOWN and event.key == pygame.K_ESCAPE:
                state = GameState.PLAYING

        elif state == GameState.GAME_OVER:
            if event.type == pygame.KEYDOWN and event.key == pygame.K_r:
                reset_game()
                state = GameState.PLAYING

    # Update
    if state == GameState.PLAYING:
        all_sprites.update(dt)
        check_collisions()

    # Draw
    screen.fill((20, 20, 35))

    if state == GameState.MENU:
        draw_menu(screen)
    elif state == GameState.PLAYING:
        draw_game(screen)
    elif state == GameState.PAUSED:
        draw_game(screen)           # Draw game behind pause overlay
        draw_pause_overlay(screen)
    elif state == GameState.GAME_OVER:
        draw_game_over(screen)

    pygame.display.flip()
```

## Scene Manager

For larger projects, encapsulate each state as a **Scene** class:

```python
class Scene:
    """Base class for all game scenes."""

    def handle_event(self, event): pass
    def update(self, dt: float):   pass
    def draw(self, screen):        pass
    def on_enter(self):            pass
    def on_exit(self):             pass


class SceneManager:
    """Manages the active scene and transitions between scenes."""

    def __init__(self):
        self._scenes: dict[str, Scene] = {}
        self._current: Scene | None    = None
        self._current_name: str        = ""

    def register(self, name: str, scene: Scene):
        self._scenes[name] = scene

    def switch(self, name: str):
        if self._current:
            self._current.on_exit()
        self._current_name = name
        self._current      = self._scenes[name]
        self._current.on_enter()

    def handle_event(self, event):
        if self._current:
            self._current.handle_event(event)

    def update(self, dt: float):
        if self._current:
            self._current.update(dt)

    def draw(self, screen):
        if self._current:
            self._current.draw(screen)
```

```python
class MenuScene(Scene):
    def __init__(self, manager: SceneManager):
        self.manager = manager
        self.font    = pygame.font.SysFont(None, 64)
        self.small   = pygame.font.SysFont(None, 30)

    def handle_event(self, event):
        if event.type == pygame.KEYDOWN and event.key == pygame.K_RETURN:
            self.manager.switch("game")

    def draw(self, screen):
        screen.fill((15, 15, 30))
        title = self.font.render("MY GAME", True, (255, 220, 0))
        start = self.small.render("Press ENTER to start", True, (200, 200, 200))
        screen.blit(title, title.get_rect(center=(screen.get_width()//2, 200)))
        screen.blit(start, start.get_rect(center=(screen.get_width()//2, 300)))


class PauseScene(Scene):
    def __init__(self, manager: SceneManager, game_scene):
        self.manager    = manager
        self.game_scene = game_scene
        self.font       = pygame.font.SysFont(None, 52)
        self.small      = pygame.font.SysFont(None, 28)
        self.overlay    = pygame.Surface(
            pygame.display.get_window_size(), pygame.SRCALPHA
        )
        self.overlay.fill((0, 0, 0, 160))

    def handle_event(self, event):
        if event.type == pygame.KEYDOWN:
            if event.key == pygame.K_ESCAPE:
                self.manager.switch("game")
            if event.key == pygame.K_q:
                self.manager.switch("menu")

    def draw(self, screen):
        # Render game behind the overlay
        self.game_scene.draw(screen)
        screen.blit(self.overlay, (0, 0))
        w, h = screen.get_width(), screen.get_height()
        pause = self.font.render("PAUSED",         True, (255, 255, 255))
        hint  = self.small.render("ESC=resume  Q=menu", True, (180, 180, 180))
        screen.blit(pause, pause.get_rect(center=(w//2, h//2 - 30)))
        screen.blit(hint,  hint.get_rect(center=(w//2, h//2 + 30)))
```

## Complete Mini-Game with Full Architecture

The following is a self-contained mini-game demonstrating all architecture patterns from this guide:

```python
import pygame
import sys
import random
import math
from enum import Enum, auto

# ── Constants ────────────────────────────────────────────────
W, H   = 800, 600
FPS    = 60
TITLE  = "Mini Shooter"

# Colors
BG     = (12,  12,  25)
WHITE  = (255, 255, 255)
YELLOW = (255, 220,  50)
RED    = (220,  60,  60)
CYAN   = ( 80, 200, 255)
GREEN  = ( 80, 220, 100)
GRAY   = (100, 100, 120)

# ── Game State Enum ──────────────────────────────────────────
class State(Enum):
    MENU      = auto()
    PLAYING   = auto()
    PAUSED    = auto()
    GAME_OVER = auto()

# ── Sprites ──────────────────────────────────────────────────
class Bullet(pygame.sprite.Sprite):
    def __init__(self, x, y):
        super().__init__()
        self.image = pygame.Surface((4, 12), pygame.SRCALPHA)
        pygame.draw.rect(self.image, YELLOW, (0, 0, 4, 12), border_radius=2)
        self.rect = self.image.get_rect(centerx=x, bottom=y)
        self.vy   = -9

    def update(self, dt):
        self.rect.y += self.vy
        if self.rect.bottom < 0:
            self.kill()


class Enemy(pygame.sprite.Sprite):
    def __init__(self, x, y, speed=1.5):
        super().__init__()
        self.image  = pygame.Surface((32, 24), pygame.SRCALPHA)
        pygame.draw.ellipse(self.image, RED, (0, 4, 32, 16))
        pygame.draw.rect(self.image, (180, 40, 40), (12, 0, 8, 8))
        self.rect   = self.image.get_rect(center=(x, y))
        self.speed  = speed
        self.angle  = random.uniform(0, math.pi * 2)
        self.health = 1

    def update(self, dt):
        self.angle += 0.03
        self.rect.x += int(math.cos(self.angle) * self.speed)
        self.rect.y += int(self.speed * 0.4)
        if self.rect.top > H + 20:
            self.kill()


class Player(pygame.sprite.Sprite):
    SPEED    = 280   # px/sec
    COOLDOWN = 0.18  # sec between shots

    def __init__(self):
        super().__init__()
        self.image = pygame.Surface((44, 34), pygame.SRCALPHA)
        pygame.draw.polygon(self.image, CYAN, [(22, 0), (0, 34), (44, 34)])
        pygame.draw.polygon(self.image, (40, 120, 180), [(22, 0), (0, 34), (44, 34)], 2)
        self.rect      = self.image.get_rect(midbottom=(W//2, H - 20))
        self.health    = 5
        self._cooldown = 0.0

    def update(self, dt):
        keys = pygame.key.get_pressed()
        dx   = (keys[pygame.K_RIGHT] or keys[pygame.K_d]) - \
               (keys[pygame.K_LEFT]  or keys[pygame.K_a])
        self.rect.x += int(dx * self.SPEED * dt)
        self.rect.clamp_ip(pygame.Rect(0, 0, W, H))
        if self._cooldown > 0:
            self._cooldown -= dt

    def can_shoot(self):
        return self._cooldown <= 0

    def shoot(self):
        self._cooldown = self.COOLDOWN
        return Bullet(self.rect.centerx, self.rect.top)


# ── HUD ──────────────────────────────────────────────────────
class HUD:
    def __init__(self):
        self.big   = pygame.font.SysFont("monospace", 22, bold=True)
        self.small = pygame.font.SysFont("monospace", 16)

    def draw(self, screen, score, health, wave):
        score_s = self.big.render(f"SCORE  {score:06d}", True, YELLOW)
        health_s = self.small.render(
            "HP: " + "♥ " * health + "♡ " * (5 - health), True, RED
        )
        wave_s   = self.small.render(f"WAVE {wave}", True, GRAY)
        screen.blit(score_s,  (10, 10))
        screen.blit(health_s, (10, 40))
        screen.blit(wave_s,   (W - wave_s.get_width() - 10, 10))


# ── Game Class ───────────────────────────────────────────────
class Game:
    def __init__(self):
        pygame.init()
        self.screen = pygame.display.set_mode((W, H))
        pygame.display.set_caption(TITLE)
        self.clock  = pygame.time.Clock()
        self.font_l = pygame.font.SysFont("monospace", 56, bold=True)
        self.font_m = pygame.font.SysFont("monospace", 26)
        self.font_s = pygame.font.SysFont("monospace", 18)
        self.hud    = HUD()
        self.state  = State.MENU
        self._reset()

    def _reset(self):
        self.all_sprites  = pygame.sprite.Group()
        self.enemies      = pygame.sprite.Group()
        self.bullets      = pygame.sprite.Group()
        self.player       = Player()
        self.all_sprites.add(self.player)
        self.score        = 0
        self.wave         = 1
        self.wave_timer   = 0.0
        self.wave_interval = 3.0
        self.enemy_speed  = 1.5
        self._spawn_wave()

    def _spawn_wave(self):
        cols   = min(3 + self.wave, 10)
        speed  = self.enemy_speed + self.wave * 0.15
        for i in range(cols):
            x = 60 + i * (W - 120) // max(cols - 1, 1)
            e = Enemy(x, -30, speed=speed)
            self.all_sprites.add(e)
            self.enemies.add(e)

    def _handle_events(self):
        for event in pygame.event.get():
            if event.type == pygame.QUIT:
                pygame.quit()
                sys.exit()

            if event.type == pygame.KEYDOWN:
                key = event.key

                if self.state == State.MENU:
                    if key == pygame.K_RETURN:
                        self.state = State.PLAYING

                elif self.state == State.PLAYING:
                    if key == pygame.K_ESCAPE:
                        self.state = State.PAUSED

                elif self.state == State.PAUSED:
                    if key == pygame.K_ESCAPE:
                        self.state = State.PLAYING
                    if key == pygame.K_q:
                        self.state = State.MENU

                elif self.state == State.GAME_OVER:
                    if key == pygame.K_r:
                        self._reset()
                        self.state = State.PLAYING
                    if key == pygame.K_q:
                        self.state = State.MENU

    def _update(self, dt):
        if self.state != State.PLAYING:
            return

        self.all_sprites.update(dt)

        # Shoot
        keys = pygame.key.get_pressed()
        if (keys[pygame.K_SPACE] or keys[pygame.K_z]) and self.player.can_shoot():
            b = self.player.shoot()
            self.all_sprites.add(b)
            self.bullets.add(b)

        # Bullets vs enemies
        hits = pygame.sprite.groupcollide(
            self.enemies, self.bullets,
            dokilla=True, dokillb=True,
        )
        self.score += len(hits) * 100

        # Enemies vs player
        if pygame.sprite.spritecollide(self.player, self.enemies, dokill=True):
            self.player.health -= 1
            if self.player.health <= 0:
                self.state = State.GAME_OVER
                return

        # Next wave
        self.wave_timer += dt
        if not self.enemies or self.wave_timer >= self.wave_interval:
            self.wave_timer = 0.0
            self.wave      += 1
            self._spawn_wave()

    def _draw_starfield(self):
        """Simple parallax star background."""
        rng = random.Random(42)
        t   = pygame.time.get_ticks()
        for _ in range(80):
            x  = rng.randint(0, W)
            y  = (rng.randint(0, H) + t * rng.uniform(0.01, 0.04)) % H
            r  = rng.randint(1, 2)
            br = rng.randint(120, 220)
            pygame.draw.circle(self.screen, (br, br, br), (int(x), int(y)), r)

    def _draw(self):
        self.screen.fill(BG)
        self._draw_starfield()

        if self.state == State.MENU:
            title = self.font_l.render(TITLE.upper(), True, YELLOW)
            sub   = self.font_m.render("ENTER = start", True, WHITE)
            self.screen.blit(title, title.get_rect(center=(W//2, H//2 - 60)))
            self.screen.blit(sub,   sub.get_rect(center=(W//2,   H//2 + 20)))
            return

        # Playing / paused / game-over share the same base render
        self.all_sprites.draw(self.screen)
        self.hud.draw(self.screen, self.score, self.player.health, self.wave)

        if self.state == State.PAUSED:
            ov = pygame.Surface((W, H), pygame.SRCALPHA)
            ov.fill((0, 0, 0, 150))
            self.screen.blit(ov, (0, 0))
            paused = self.font_l.render("PAUSED", True, WHITE)
            hint   = self.font_s.render("ESC=resume   Q=menu", True, GRAY)
            self.screen.blit(paused, paused.get_rect(center=(W//2, H//2 - 40)))
            self.screen.blit(hint,   hint.get_rect(center=(W//2,   H//2 + 30)))

        elif self.state == State.GAME_OVER:
            ov = pygame.Surface((W, H), pygame.SRCALPHA)
            ov.fill((0, 0, 0, 170))
            self.screen.blit(ov, (0, 0))
            go     = self.font_l.render("GAME OVER",     True, RED)
            sc     = self.font_m.render(f"Score: {self.score:06d}", True, YELLOW)
            hint   = self.font_s.render("R=restart   Q=menu",      True, GRAY)
            self.screen.blit(go,   go.get_rect(center=(W//2,   H//2 - 60)))
            self.screen.blit(sc,   sc.get_rect(center=(W//2,   H//2)))
            self.screen.blit(hint, hint.get_rect(center=(W//2, H//2 + 50)))

        pygame.display.flip()

    def run(self):
        while True:
            dt = self.clock.tick(FPS) / 1000.0
            self._handle_events()
            self._update(dt)
            self._draw()


# ── Entry Point ──────────────────────────────────────────────
if __name__ == "__main__":
    Game().run()
```

## Performance Tips

### Dirty Rect Updates

```python
# Instead of pygame.display.flip() (redraws entire screen):
dirty = all_sprites.draw(screen)      # RenderUpdates returns dirty rects
pygame.display.update(dirty)          # Only update changed regions
```

### Pre-convert Surfaces

```python
# After loading any image — do this ONCE, not every frame
image = pygame.image.load("sprite.png").convert_alpha()

# For opaque images (no transparency)
background = pygame.image.load("bg.png").convert()
```

### Avoid Per-Pixel Operations in the Loop

```python
# SLOW — never do this in the game loop
for x in range(800):
    for y in range(600):
        screen.set_at((x, y), (255, 0, 0))

# FAST — use draw functions or pre-built surfaces
screen.fill((255, 0, 0))
```

### Cache Rendered Text

```python
# SLOW — re-renders every frame even when unchanged
def draw_hud():
    surf = font.render(f"Score: {score}", True, WHITE)
    screen.blit(surf, (10, 10))

# FAST — only re-render when value changes
_score_cache = {"value": None, "surf": None}

def draw_hud():
    if score != _score_cache["value"]:
        _score_cache["value"] = score
        _score_cache["surf"]  = font.render(f"Score: {score}", True, WHITE)
    screen.blit(_score_cache["surf"], (10, 10))
```

### Group Draw Order

```python
# Draw back-to-front (painter's algorithm)
background_group.draw(screen)
terrain_group.draw(screen)
enemy_group.draw(screen)
player_group.draw(screen)
bullet_group.draw(screen)
particle_group.draw(screen)
ui_group.draw(screen)
```

:::tip[Profile Before Optimizing]
Use Python's `cProfile` to find real bottlenecks before applying optimizations:
```bash
python -m cProfile -s cumtime main.py | head -30
```
In most Pygame games, the bottleneck is rendering (blit calls and display.flip), not Python logic.
:::

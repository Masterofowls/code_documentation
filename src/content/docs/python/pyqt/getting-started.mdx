## Installation

<Steps>

1. **Create and activate a virtual environment**

   ```bash
   python -m venv .venv
   # Windows
   .venv\Scripts\activate
   # macOS / Linux
   source .venv/bin/activate
   ```

2. **Install PyQt6**

   ```bash
   pip install PyQt6
   ```

3. **Optional extras**

   ```bash
   # Qt Designer + pyuic6 / pyrcc6 tools
   pip install PyQt6-tools

   # Charting module
   pip install PyQt6-Charts
   ```

4. **Verify the installation**

   ```python
   python -c "from PyQt6.QtWidgets import QApplication; print('PyQt6 OK')"
   ```

</Steps>

:::caution[Python version]
PyQt6 requires **Python 3.6.1 or later**. Python 3.10+ is recommended for the best type-hint and IDE support.
:::

## The Three Pillars of Every PyQt6 App

### 1. QApplication

`QApplication` is the global application object. There must be **exactly one** per process — create it before any widget. It owns the event loop, manages the application lifecycle, and forwards system events to the right widget.

```python
import sys
from PyQt6.QtWidgets import QApplication

app = QApplication(sys.argv)   # parse Qt's own flags from argv
# ... create windows ...
sys.exit(app.exec())           # start the event loop; blocks until app quits
```

| Useful QApplication methods            | Purpose                                          |
| -------------------------------------- | ------------------------------------------------ |
| `app.setApplicationName("Name")`       | Sets the app name (used in title bars, settings) |
| `app.setOrganizationName("Org")`       | Used by QSettings for config storage             |
| `app.setWindowIcon(QIcon("icon.png"))` | Default icon for all windows                     |
| `app.setStyleSheet("...")`             | Global QSS applied to every widget               |
| `app.quit()`                           | Request a clean exit                             |
| `app.instance()`                       | Return the existing QApplication from anywhere   |

### 2. QMainWindow

`QMainWindow` is the top-level window class with built-in slots for a **menu bar**, **toolbars**, a **central widget**, **dock widgets**, and a **status bar**.

```
┌──────────────────────────────────────┐
│ Menu Bar                             │
├──────────────────────────────────────┤
│ Tool Bar                             │
├──────────┬───────────────────────────┤
│ Dock     │   Central Widget          │
│ Widget   │                           │
├──────────┴───────────────────────────┤
│ Status Bar                           │
└──────────────────────────────────────┘
```

Every `QMainWindow` **must** have a central widget set via `setCentralWidget()`.

### 3. QWidget

`QWidget` is the base class for all UI elements. When used as a standalone window (no parent) it acts as a plain window. As the central widget it becomes the canvas you arrange other widgets on.

## Building Your First App Step by Step

<Steps>

1. **Minimal skeleton**

   ```python
   import sys
   from PyQt6.QtWidgets import QApplication, QMainWindow

   class MainWindow(QMainWindow):
       def __init__(self) -> None:
           super().__init__()
           self.setWindowTitle("My First App")
           self.resize(640, 480)

   app = QApplication(sys.argv)
   win = MainWindow()
   win.show()
   sys.exit(app.exec())
   ```

2. **Add a central widget with a layout**

   ```python
   from PyQt6.QtWidgets import QWidget, QVBoxLayout, QLabel

   class MainWindow(QMainWindow):
       def __init__(self) -> None:
           super().__init__()
           self.setWindowTitle("My First App")
           self.resize(640, 480)

           container = QWidget()
           layout = QVBoxLayout(container)

           label = QLabel("Welcome to PyQt6!")
           layout.addWidget(label)

           self.setCentralWidget(container)
   ```

3. **Add a QPushButton with a click handler**

   ```python
   from PyQt6.QtWidgets import QPushButton

   class MainWindow(QMainWindow):
       def __init__(self) -> None:
           super().__init__()
           self.setWindowTitle("My First App")
           self.resize(640, 480)

           container = QWidget()
           layout = QVBoxLayout(container)

           self.label = QLabel("Click the button!")
           layout.addWidget(self.label)

           button = QPushButton("Click Me")
           button.clicked.connect(self.on_button_clicked)
           layout.addWidget(button)

           self.setCentralWidget(container)

       def on_button_clicked(self) -> None:
           self.label.setText("Button was clicked! ✅")
   ```

4. **Exit cleanly**

   Connect the quit button to `QApplication.instance().quit()` or use the built-in window close button — Qt handles `closeEvent` automatically.

   ```python
   quit_btn = QPushButton("Quit")
   quit_btn.clicked.connect(QApplication.instance().quit)
   layout.addWidget(quit_btn)
   ```

</Steps>

## Full Working Example

```python
import sys
from PyQt6.QtWidgets import (
    QApplication,
    QMainWindow,
    QWidget,
    QVBoxLayout,
    QHBoxLayout,
    QLabel,
    QPushButton,
    QLineEdit,
)
from PyQt6.QtCore import Qt
from PyQt6.QtGui import QFont


class MainWindow(QMainWindow):
    """First complete PyQt6 application."""

    def __init__(self) -> None:
        super().__init__()
        self.setWindowTitle("PyQt6 Getting Started")
        self.setMinimumSize(500, 300)

        self._click_count = 0

        # ── Central container ──────────────────────────────────────
        container = QWidget()
        root = QVBoxLayout(container)
        root.setSpacing(12)
        root.setContentsMargins(20, 20, 20, 20)

        # Title label
        title = QLabel("Hello, PyQt6!")
        title.setFont(QFont("Arial", 20, QFont.Weight.Bold))
        title.setAlignment(Qt.AlignmentFlag.AlignHCenter)
        root.addWidget(title)

        # Name input row
        input_row = QHBoxLayout()
        input_row.addWidget(QLabel("Your name:"))
        self.name_input = QLineEdit()
        self.name_input.setPlaceholderText("Type your name here…")
        input_row.addWidget(self.name_input)
        root.addLayout(input_row)

        # Greet button
        greet_btn = QPushButton("Greet Me")
        greet_btn.setFixedHeight(36)
        greet_btn.clicked.connect(self.greet)
        root.addWidget(greet_btn)

        # Result label
        self.result_label = QLabel("")
        self.result_label.setAlignment(Qt.AlignmentFlag.AlignHCenter)
        self.result_label.setStyleSheet("font-size: 16px; color: #16a34a;")
        root.addWidget(self.result_label)

        # Counter row
        counter_row = QHBoxLayout()
        dec_btn = QPushButton("−")
        dec_btn.clicked.connect(self.decrement)
        inc_btn = QPushButton("+")
        inc_btn.clicked.connect(self.increment)
        self.counter_label = QLabel("Count: 0")
        self.counter_label.setAlignment(Qt.AlignmentFlag.AlignHCenter)
        counter_row.addWidget(dec_btn)
        counter_row.addWidget(self.counter_label)
        counter_row.addWidget(inc_btn)
        root.addLayout(counter_row)

        root.addStretch()  # push everything to the top

        # Quit button at bottom
        quit_btn = QPushButton("Quit")
        quit_btn.setStyleSheet("background-color: #ef4444; color: white;")
        quit_btn.clicked.connect(QApplication.instance().quit)
        root.addWidget(quit_btn)

        self.setCentralWidget(container)

        # Status bar feedback
        self.statusBar().showMessage("Ready")

    # ── Slot methods ───────────────────────────────────────────────

    def greet(self) -> None:
        name = self.name_input.text().strip() or "stranger"
        self.result_label.setText(f"Hello, {name}! 👋")
        self.statusBar().showMessage(f"Greeted {name}")

    def increment(self) -> None:
        self._click_count += 1
        self.counter_label.setText(f"Count: {self._click_count}")

    def decrement(self) -> None:
        self._click_count -= 1
        self.counter_label.setText(f"Count: {self._click_count}")


def main() -> None:
    app = QApplication(sys.argv)
    app.setApplicationName("GettingStarted")
    window = MainWindow()
    window.show()
    sys.exit(app.exec())


if __name__ == "__main__":
    main()
```

## Understanding the Event Loop

When you call `app.exec()`, Qt enters a loop that:

1. **Waits** for events (mouse clicks, key presses, timers, socket data…).
2. **Dispatches** each event to the appropriate widget or handler.
3. **Repaints** widgets that have been marked dirty.
4. **Repeats** until `quit()` is called or the last window closes.

```
app.exec()
  └── event queue
        ├── MousePressEvent → widget.mousePressEvent()
        ├── KeyPressEvent   → focusedWidget.keyPressEvent()
        ├── TimerEvent      → QTimer.timeout signal
        └── PaintEvent      → widget.paintEvent()
```

:::caution[Never block the event loop]
Running long operations (file I/O, network calls, heavy computation) directly in a slot **freezes the UI**. Move slow work to a `QThread` or use `QTimer.singleShot()` for deferred tasks. See the [Signals & Slots](/python/pyqt/signals-slots) page for the worker-thread pattern.
:::

## Window Geometry & State

```python
# Set size
window.resize(800, 600)
window.setMinimumSize(400, 300)
window.setMaximumSize(1920, 1080)
window.setFixedSize(640, 480)   # prevents resizing entirely

# Position
window.move(100, 100)           # screen coordinates
window.center()                  # not built-in — implement manually:

def center_window(win: QMainWindow) -> None:
    screen = QApplication.primaryScreen().geometry()
    frame = win.frameGeometry()
    frame.moveCenter(screen.center())
    win.move(frame.topLeft())

# Save/restore geometry between sessions
from PyQt6.QtCore import QSettings

settings = QSettings()

def closeEvent(self, event):
    settings.setValue("geometry", self.saveGeometry())
    super().closeEvent(event)

def __init__(self):
    ...
    if saved := settings.value("geometry"):
        self.restoreGeometry(saved)
```

## Qt Designer Workflow

Qt Designer lets you build UIs visually and save them as `.ui` XML files.

```bash
# Launch Qt Designer (requires PyQt6-tools)
pyqt6-tools designer
```

Load a `.ui` file at runtime (no code generation needed):

```python
from PyQt6 import uic
from PyQt6.QtWidgets import QMainWindow

class MainWindow(QMainWindow):
    def __init__(self) -> None:
        super().__init__()
        uic.loadUi("ui/main_window.ui", self)
        # All named widgets are now attributes: self.pushButton, self.lineEdit, etc.
        self.pushButton.clicked.connect(self.handle_click)

    def handle_click(self) -> None:
        print(self.lineEdit.text())
```

Alternatively, convert to Python code once:

```bash
pyuic6 -x main_window.ui -o ui_main_window.py
```

## Recommended Project Structure

```
my_app/
├── main.py                  # QApplication + entry point
├── main_window.py           # MainWindow class
├── ui/
│   └── main_window.ui       # Qt Designer source
├── widgets/                 # Custom reusable widgets
├── models/                  # Data / business logic
├── resources/
│   ├── icons/
│   └── app.qrc              # Qt resource file
├── tests/
└── requirements.txt
```

:::tip[Separate UI from logic]
Keep slot methods thin — call into a separate model or service layer. This makes unit-testing your logic trivial without needing a display.
:::

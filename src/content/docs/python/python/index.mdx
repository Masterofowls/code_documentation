Python is a high-level, interpreted, dynamically typed programming language renowned for its clean syntax and readability. Created by Guido van Rossum and first released in 1991, Python has grown into one of the most popular languages in the world — used in web development, data science, artificial intelligence, automation, and more.

## Python Versions

Always use the latest stable Python 3 release. Python 2 reached end-of-life in January 2020 and **should not be used** for new projects.

| Version | Status |
|---------|--------|
| 3.13 | ✅ Latest stable |
| 3.12 | ✅ Supported |
| 3.11 | ✅ Supported |
| 3.10 | ✅ Supported |
| 2.x | ❌ End of Life |

## Quick Taste

```python
# Python is expressive and readable
names = ['Alice', 'Bob', 'Carol']

for name in names:
    greeting = f'Hello, {name}!'
    print(greeting)
```

```
Hello, Alice!
Hello, Bob!
Hello, Carol!
```

:::tip[Choosing a Version]
Use [pyenv](https://github.com/pyenv/pyenv) to manage multiple Python versions on your machine without conflicts.
:::

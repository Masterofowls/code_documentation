PowerShell is a cross-platform task automation solution made up of a command-line shell, a scripting language, and a configuration management framework. It runs on Windows, Linux, and macOS.

## Why PowerShell?

- **Object-based pipeline** — outputs .NET objects, not plain text
- **Cross-platform** — PowerShell 7+ runs on Windows, macOS, Linux
- **Rich scripting** — full programming language with modules, classes, and error handling
- **Deep Windows integration** — manages Windows services, registry, AD, and Azure

## Quick Start

```powershell
# Check PowerShell version
$PSVersionTable.PSVersion

# List files
Get-ChildItem -Path . -Filter "*.ps1"

# Run a script
.\script.ps1
```

`llama.cpp` is a high-performance C++ inference engine for running large language models (LLMs) locally without a GPU, using quantization to reduce memory requirements dramatically.

## Why llama.cpp?

- **CPU inference** — run LLMs on any machine, no GPU required
- **Quantization** — 4-bit quantized models fit in 4–8 GB RAM
- **Blazing performance** — hand-optimized SIMD (AVX2/NEON) kernels
- **Ecosystem** — powers Ollama, LM Studio, and many local AI tools

## Quick Start

```bash
# Clone and build
git clone https://github.com/ggerganov/llama.cpp
cd llama.cpp
cmake -B build && cmake --build build -j8

# Download a model (example: Llama 3.2 3B GGUF)
# Then run inference
./build/bin/llama-cli -m model.gguf -p "Explain quantum computing" -n 256
```

## Basic C++ Integration

```cpp
#include "llama.h"

int main() {
    llama_model_params model_params = llama_model_default_params();
    llama_model* model = llama_model_load_from_file("model.gguf", model_params);

    llama_context_params ctx_params = llama_context_default_params();
    llama_context* ctx = llama_init_from_model(model, ctx_params);

    // tokenize and run inference...

    llama_free(ctx);
    llama_model_free(model);
    return 0;
}
```

Asynchronous code is everywhere in modern JavaScript — API calls, database queries, file I/O, timers, and event streams. Jest provides several patterns for testing async code correctly and reliably.

## The Problem with Async Tests

Without proper async handling, tests can pass incorrectly:

```typescript
// WRONG — this test always passes because Jest doesn't wait for the assertion
test('fetches user data', () => {
  fetchUser(1).then((user) => {
    expect(user.name).toBe('Alice'); // never executed if the test ends first
  });
});
```

Jest finishes the test synchronously, the assertion inside `.then()` never runs, and the test passes despite the potential failure.

## Callbacks with `done`

For older callback-based APIs, use the `done` parameter. Jest waits until `done()` is called before considering the test complete.

```typescript
import { readFileCallback } from './fileReader';

test('reads file contents', (done) => {
  readFileCallback('./data.txt', (error, contents) => {
    if (error) {
      done(error); // fail the test with the error
      return;
    }
    expect(contents).toContain('Hello');
    done(); // signal the test is complete
  });
});
```

:::caution[Always call done on error paths]
If you forget to call `done(error)` on the error path, the test will time out rather than fail with a useful message. Always handle both the success and error cases.
:::

### Converting callbacks to Promises

When possible, wrap callback APIs in Promises for cleaner tests:

```typescript
import { promisify } from 'util';
import { readFile } from 'fs';

const readFileAsync = promisify(readFile);

test('reads file contents', async () => {
  const contents = await readFileAsync('./data.txt', 'utf8');
  expect(contents).toContain('Hello');
});
```

## Testing Promises

### Return the promise

The simplest way to test Promises is to return them from the test. Jest will wait for the returned Promise to settle.

```typescript
test('resolves with user data', () => {
  return fetchUser(1).then((user) => {
    expect(user.name).toBe('Alice');
    expect(user.email).toBeDefined();
  });
});

test('rejects on invalid ID', () => {
  return expect(fetchUser(-1)).rejects.toThrow('Invalid user ID');
});
```

### `.resolves` and `.rejects` matchers

These matchers unwrap the Promise before running the assertion:

```typescript
test('resolves with user data', async () => {
  await expect(fetchUser(1)).resolves.toEqual(
    expect.objectContaining({ name: 'Alice' }),
  );
});

test('rejects with a network error', async () => {
  await expect(fetchFromBrokenEndpoint()).rejects.toThrow('Network error');
  await expect(fetchFromBrokenEndpoint()).rejects.toMatchObject({
    message: expect.stringContaining('Network'),
    status: 503,
  });
});
```

:::tip[Always await .resolves and .rejects]
`expect(promise).resolves.toBe(value)` returns a Promise. If you forget to `await` it, the assertion never runs and the test passes regardless of the actual resolved value.
:::

## `async` / `await`

The cleanest pattern for testing Promises is `async/await`:

```typescript
test('creates and retrieves a user', async () => {
  // Arrange
  const payload = { name: 'Alice', email: 'alice@example.com' };

  // Act
  const created = await userService.create(payload);
  const retrieved = await userService.findById(created.id);

  // Assert
  expect(retrieved.name).toBe('Alice');
  expect(retrieved.email).toBe('alice@example.com');
});
```

### Testing async errors with try/catch

```typescript
test('throws when user is not found', async () => {
  await expect(userService.findById(99999)).rejects.toThrow('User not found');
});

// Alternatively with try/catch (more verbose but useful for inspecting the error)
test('throws with correct error code', async () => {
  try {
    await userService.findById(99999);
    fail('Expected an error to be thrown'); // force failure if no error thrown
  } catch (error) {
    expect(error).toBeInstanceOf(NotFoundError);
    expect((error as NotFoundError).code).toBe('USER_NOT_FOUND');
    expect((error as NotFoundError).statusCode).toBe(404);
  }
});
```

## `expect.assertions()` — Guarding Async Error Paths

`expect.assertions(N)` declares that exactly N assertions must be called during the test. If fewer assertions run (e.g., because a Promise resolved instead of rejecting), the test fails.

```typescript
test('rejects on database error', async () => {
  expect.assertions(2); // exactly 2 assertions must run

  mockDb.query.mockRejectedValue(new Error('Connection refused'));

  try {
    await userService.findById(1);
  } catch (error) {
    expect(error).toBeInstanceOf(DatabaseError); // assertion 1
    expect((error as DatabaseError).message).toContain('Connection refused'); // assertion 2
  }
});
```

Without `expect.assertions(2)`, if `findById` unexpectedly resolves, the `catch` block never runs, the test has zero assertions, and **Jest considers it a pass**.

Use `expect.hasAssertions()` when you know at least one assertion should run, but you don't know exactly how many:

```typescript
test('processes all items', async () => {
  expect.hasAssertions();

  const items = await fetchItems();
  items.forEach((item) => {
    expect(item).toHaveProperty('id');
  });
});
```

## Testing `setTimeout` and `setInterval` with Fake Timers

```typescript
// src/utils/debounce.ts
export function debounce<T extends (...args: unknown[]) => void>(
  fn: T,
  delay: number,
): T {
  let timer: ReturnType<typeof setTimeout>;
  return function (this: unknown, ...args: Parameters<T>) {
    clearTimeout(timer);
    timer = setTimeout(() => fn.apply(this, args), delay);
  } as T;
}
```

```typescript
// src/utils/debounce.test.ts
import { debounce } from './debounce';

describe('debounce', () => {
  beforeEach(() => {
    jest.useFakeTimers();
  });

  afterEach(() => {
    jest.useRealTimers();
  });

  it('calls the function after the delay', () => {
    const mockFn = jest.fn();
    const debounced = debounce(mockFn, 300);

    debounced('call 1');
    expect(mockFn).not.toHaveBeenCalled(); // not called yet

    jest.advanceTimersByTime(300);
    expect(mockFn).toHaveBeenCalledTimes(1);
    expect(mockFn).toHaveBeenCalledWith('call 1');
  });

  it('only calls the function once for rapid calls', () => {
    const mockFn = jest.fn();
    const debounced = debounce(mockFn, 300);

    debounced('call 1');
    debounced('call 2');
    debounced('call 3');

    jest.advanceTimersByTime(300);

    expect(mockFn).toHaveBeenCalledTimes(1);
    expect(mockFn).toHaveBeenCalledWith('call 3'); // last call wins
  });
});
```

### Testing polling with `setInterval`

```typescript
// src/utils/poller.ts
export class StatusPoller {
  private intervalId: ReturnType<typeof setInterval> | null = null;

  start(checkFn: () => Promise<void>, intervalMs: number) {
    this.intervalId = setInterval(checkFn, intervalMs);
  }

  stop() {
    if (this.intervalId) {
      clearInterval(this.intervalId);
      this.intervalId = null;
    }
  }
}
```

```typescript
describe('StatusPoller', () => {
  beforeEach(() => jest.useFakeTimers());
  afterEach(() => jest.useRealTimers());

  it('calls checkFn at the specified interval', async () => {
    const checkFn = jest.fn().mockResolvedValue(undefined);
    const poller = new StatusPoller();

    poller.start(checkFn, 1000);

    jest.advanceTimersByTime(3500);

    // Called at t=1000, t=2000, t=3000
    expect(checkFn).toHaveBeenCalledTimes(3);

    poller.stop();
  });
});
```

## Testing `fetch`

```typescript
// src/services/weatherService.ts
export async function getWeather(city: string) {
  const response = await fetch(`https://api.weather.com/v1/current?q=${city}`);
  if (!response.ok) {
    throw new Error(`Failed to fetch weather: ${response.status}`);
  }
  return response.json();
}
```

```typescript
// src/services/weatherService.test.ts
global.fetch = jest.fn();

describe('getWeather', () => {
  beforeEach(() => {
    jest.clearAllMocks();
  });

  it('returns weather data for a valid city', async () => {
    const mockData = { city: 'Paris', temp: 22, condition: 'Sunny' };
    (global.fetch as jest.Mock).mockResolvedValue({
      ok: true,
      json: jest.fn().mockResolvedValue(mockData),
    });

    const result = await getWeather('Paris');

    expect(global.fetch).toHaveBeenCalledWith(
      'https://api.weather.com/v1/current?q=Paris',
    );
    expect(result).toEqual(mockData);
  });

  it('throws on non-OK response', async () => {
    (global.fetch as jest.Mock).mockResolvedValue({
      ok: false,
      status: 404,
    });

    await expect(getWeather('Atlantis')).rejects.toThrow(
      'Failed to fetch weather: 404',
    );
  });

  it('propagates network errors', async () => {
    (global.fetch as jest.Mock).mockRejectedValue(
      new TypeError('Failed to fetch'),
    );

    await expect(getWeather('Paris')).rejects.toThrow('Failed to fetch');
  });
});
```

## Testing Axios

```typescript
// src/services/userApiService.ts
import axios from 'axios';

export async function getUser(id: number) {
  const { data } = await axios.get(`/api/users/${id}`);
  return data;

export async function createUser(payload: { name: string; email: string }) {
  const { data } = await axios.post('/api/users', payload);
```

```typescript
// src/services/userApiService.test.ts
import axios from 'axios';
import { getUser, createUser } from './userApiService';

jest.mock('axios');
const mockedAxios = axios as jest.Mocked<typeof axios>;

describe('getUser', () => {
  it('fetches user by ID', async () => {
    mockedAxios.get.mockResolvedValue({
      data: { id: 1, name: 'Alice', email: 'alice@example.com' },
    });

    const user = await getUser(1);

    expect(mockedAxios.get).toHaveBeenCalledWith('/api/users/1');
    expect(user.name).toBe('Alice');
  });

  it('throws on 404', async () => {
    const error = Object.assign(new Error('Not Found'), {
      response: { status: 404, data: { message: 'User not found' } },
    });
    mockedAxios.get.mockRejectedValue(error);

    await expect(getUser(999)).rejects.toThrow('Not Found');
  });
});
```

```typescript
// src/events/eventBus.test.ts
import { OrderEventBus } from './eventBus';

test('emits order:placed event when order is placed', (done) => {
  const bus = new OrderEventBus();
  const order = { id: 1, item: 'Widget', quantity: 2 };

  bus.on('order:placed', (emittedOrder) => {
    expect(emittedOrder).toEqual(order);
    done();
  });

  bus.placeOrder(order);

// Using a mock listener
test('notifies all listeners', () => {
  const bus = new OrderEventBus();
  const listener = jest.fn();

  bus.on('order:placed', listener);
  bus.placeOrder({ id: 1, item: 'Widget', quantity: 2 });
  bus.placeOrder({ id: 2, item: 'Gadget', quantity: 1 });

  expect(listener).toHaveBeenCalledTimes(2);
  expect(listener).toHaveBeenNthCalledWith(
    1,
    expect.objectContaining({ item: 'Widget' }),
  );
});
```

## Retry Logic Testing

```typescript
// src/utils/withRetry.ts
export async function withRetry<T>(
  fn: () => Promise<T>,
  options: { maxRetries: number; delayMs: number },
): Promise<T> {
  let lastError: Error;
  for (let attempt = 0; attempt <= options.maxRetries; attempt++) {
    try {
      return await fn();
    } catch (error) {
      lastError = error as Error;
      if (attempt < options.maxRetries) {
        await new Promise((resolve) => setTimeout(resolve, options.delayMs));
      }
    }
  }
  throw lastError!;
}
```

```typescript
// src/utils/withRetry.test.ts
import { withRetry } from './withRetry';

describe('withRetry', () => {
  beforeEach(() => jest.useFakeTimers());
  afterEach(() => jest.useRealTimers());

  it('returns the result on first success', async () => {
    const mockFn = jest.fn().mockResolvedValue('success');

    const result = await withRetry(mockFn, { maxRetries: 3, delayMs: 100 });

    expect(result).toBe('success');
    expect(mockFn).toHaveBeenCalledTimes(1);
  });

  it('retries on failure and eventually succeeds', async () => {
    const mockFn = jest
      .fn()
      .mockRejectedValueOnce(new Error('Attempt 1 failed'))
      .mockRejectedValueOnce(new Error('Attempt 2 failed'))
      .mockResolvedValue('success on attempt 3');

    const resultPromise = withRetry(mockFn, { maxRetries: 3, delayMs: 100 });

    // Advance through the delay periods
    await Promise.resolve(); // let first attempt run
    jest.advanceTimersByTime(100);
    await Promise.resolve(); // let second attempt run
    jest.advanceTimersByTime(100);

    const result = await resultPromise;

    expect(result).toBe('success on attempt 3');
    expect(mockFn).toHaveBeenCalledTimes(3);
  });

  it('throws after exhausting all retries', async () => {
    const mockFn = jest.fn().mockRejectedValue(new Error('Always fails'));

    const resultPromise = withRetry(mockFn, { maxRetries: 2, delayMs: 50 });

    for (let i = 0; i < 2; i++) {
      await Promise.resolve();
      jest.advanceTimersByTime(50);
    }

    await expect(resultPromise).rejects.toThrow('Always fails');
    expect(mockFn).toHaveBeenCalledTimes(3); // 1 initial + 2 retries
  });
});
```

## Real-World Async Service Test

```typescript
// src/services/orderService.test.ts
import { OrderService } from './orderService';
import { paymentGateway } from './paymentGateway';
import { inventoryService } from './inventoryService';
import { emailService } from './emailService';

jest.mock('./paymentGateway');
jest.mock('./inventoryService');
jest.mock('./emailService');

const mockPayment = paymentGateway as jest.Mocked<typeof paymentGateway>;
const mockInventory = inventoryService as jest.Mocked<typeof inventoryService>;
const mockEmail = emailService as jest.Mocked<typeof emailService>;

describe('OrderService.placeOrder', () => {
  let service: OrderService;

  beforeEach(() => {
    service = new OrderService();
    jest.clearAllMocks();
  });

  it('completes a successful order', async () => {
    expect.assertions(4);

    mockInventory.reserve.mockResolvedValue({ reservationId: 'res-123' });
    mockPayment.charge.mockResolvedValue({ transactionId: 'txn-456' });
    mockEmail.send.mockResolvedValue({ messageId: 'msg-789' });

    const result = await service.placeOrder({
      userId: 1,
      items: [{ productId: 'SKU-001', quantity: 2 }],
      paymentMethod: 'card_visa',
    });

    expect(result.status).toBe('confirmed');
    expect(mockInventory.reserve).toHaveBeenCalledWith([
      { productId: 'SKU-001', quantity: 2 },
    ]);
    expect(mockPayment.charge).toHaveBeenCalledWith(
      expect.objectContaining({ userId: 1 }),
    );
    expect(mockEmail.send).toHaveBeenCalledWith(
      expect.objectContaining({ type: 'order_confirmation' }),
    );
  });

  it('releases inventory reservation when payment fails', async () => {
    mockInventory.reserve.mockResolvedValue({ reservationId: 'res-123' });
    mockPayment.charge.mockRejectedValue(new Error('Card declined'));

    await expect(
      service.placeOrder({
        userId: 1,
        items: [{ productId: 'SKU-001', quantity: 2 }],
        paymentMethod: 'card_expired',
      }),
    ).rejects.toThrow('Card declined');

    expect(mockInventory.release).toHaveBeenCalledWith('res-123');
    expect(mockEmail.send).not.toHaveBeenCalled();
  });
});
```

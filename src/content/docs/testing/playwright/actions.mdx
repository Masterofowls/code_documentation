Playwright actions are how tests interact with a page — clicking buttons, typing text, submitting forms, dragging elements, uploading files, and more. Every action has auto-waiting built in: before acting, Playwright waits for the element to be visible, enabled, stable, and receiving events.

## click

```typescript
// Basic left click
await page.getByRole('button', { name: 'Submit' }).click();

// Right click (context menu)
await page.getByRole('img', { name: 'Profile photo' }).click({ button: 'right' });

// Double click
await page.getByRole('cell', { name: 'Filename.txt' }).dblclick();

// Click at specific coordinates within element
await page.getByRole('slider').click({ position: { x: 50, y: 0 } });

// Click with modifier keys
await page.getByRole('link', { name: 'Open' }).click({ modifiers: ['Meta'] }); // Cmd+Click
await page.getByRole('row').nth(2).click({ modifiers: ['Shift'] });            // Shift+Click
await page.getByRole('row').nth(3).click({ modifiers: ['Control'] });          // Ctrl+Click

// Force click (bypass actionability checks — use sparingly)
await page.getByRole('button').click({ force: true });

// Click with custom timeout
await page.getByRole('button', { name: 'Load' }).click({ timeout: 5000 });

// No wait after click
await page.getByRole('button').click({ noWaitAfter: true });
```

## fill vs type

`fill` and `type` both enter text into inputs, but behave differently:

| | `fill` | `type` |
|---|---|---|
| Behavior | Sets the value directly | Fires key events for each character |
| Speed | Very fast | Slower (simulates keystroke timing) |
| Clears first | Yes (always clears existing value) | No (appends unless you clear first) |
| Triggers events | `input`, `change` | `keydown`, `keypress`, `keyup`, `input`, `change` |
| Use when | Most form inputs | Key event handlers, masked inputs, autocomplete |

```typescript
// fill — prefer this for most inputs
await page.getByLabel('Email').fill('alice@example.com');
await page.getByLabel('Password').fill('SecurePass123!');

// fill clears the existing value first
await page.getByRole('searchbox').fill('initial search');
await page.getByRole('searchbox').fill('updated search'); // replaces, doesn't append

// type — use when the app listens to individual key events
await page.getByLabel('Search').type('react');
// equivalent to pressing r, e, a, c, t one by one

// Combining: clear + type
await page.getByLabel('Name').clear();
await page.getByLabel('Name').type('Alice');
```

:::tip[Use fill for Speed]
`fill` is significantly faster and more reliable than `type` for standard inputs. Only use `type` when the application specifically listens to `keydown`/`keyup` events — such as keyboard shortcuts, autocomplete widgets, or masked inputs.
:::

## press

Simulate keyboard key presses:

```typescript
// Common keys
await page.getByRole('searchbox').press('Enter');
await page.getByRole('dialog').press('Escape');
await page.getByRole('textbox').press('Tab');

// Key combinations (use + separator)
await page.keyboard.press('Control+A');    // Select all
await page.keyboard.press('Control+C');    // Copy
await page.keyboard.press('Control+V');    // Paste
await page.keyboard.press('Control+Z');    // Undo
await page.keyboard.press('Meta+Shift+Z'); // Redo on Mac

// Press on a specific element
await page.getByRole('textbox').press('Control+A');
await page.getByRole('textbox').press('ArrowDown');

// Special keys
await page.keyboard.press('F5');
await page.keyboard.press('Delete');
await page.keyboard.press('Backspace');
await page.keyboard.press('Home');
await page.keyboard.press('End');
await page.keyboard.press('PageUp');
await page.keyboard.press('PageDown');
```

## check and uncheck

```typescript
// Check a checkbox
await page.getByRole('checkbox', { name: 'Subscribe to newsletter' }).check();
await expect(page.getByRole('checkbox', { name: 'Subscribe to newsletter' })).toBeChecked();

// Uncheck
await page.getByRole('checkbox', { name: 'Subscribe to newsletter' }).uncheck();
await expect(page.getByRole('checkbox', { name: 'Subscribe to newsletter' })).not.toBeChecked();

// Select a radio button
await page.getByRole('radio', { name: 'Annual billing' }).check();

// setChecked — set to specific state regardless of current state
await page.getByRole('checkbox').setChecked(true);
await page.getByRole('checkbox').setChecked(false);
```

## selectOption

Select one or more options in a `<select>` dropdown:

```typescript
// By value attribute
await page.getByRole('combobox', { name: 'Country' }).selectOption('CA');

// By visible label text
await page.getByRole('combobox', { name: 'Country' }).selectOption({ label: 'Canada' });

// By index (0-based)
await page.getByRole('combobox').selectOption({ index: 2 });

// Multiple selections (for multi-select)
await page.getByRole('listbox', { name: 'Tags' }).selectOption(['javascript', 'typescript', 'react']);

// Deselect all in multi-select
await page.getByRole('listbox').selectOption([]);
```

## hover, focus, blur

```typescript
// Hover over an element (triggers CSS :hover, shows tooltips)
await page.getByRole('button', { name: 'Options' }).hover();

// Hover at specific position
await page.getByRole('img').hover({ position: { x: 100, y: 50 } });

// Focus an element (triggers :focus styles and onFocus handlers)
await page.getByLabel('Email').focus();

// Blur — remove focus from the element
await page.getByLabel('Email').blur();

// Check focus state
await expect(page.getByLabel('Email')).toBeFocused();
```

## dragTo — Drag and Drop

```typescript
// Drag an element to another element
const source = page.getByTestId('drag-item-1');
const target = page.getByTestId('drop-zone');
await source.dragTo(target);

// Drag with options
await source.dragTo(target, {
  sourcePosition: { x: 10, y: 10 },   // start position within source element
  targetPosition: { x: 50, y: 50 },   // end position within target element
});

// Manual drag (for complex drag scenarios)
await page.mouse.move(100, 200);       // move to source
await page.mouse.down();               // press mouse button
await page.mouse.move(300, 400, { steps: 10 }); // drag with steps (smoother)
await page.mouse.up();                 // release mouse button
```

## uploadFile

```typescript
// Single file upload
await page.getByLabel('Upload photo').setInputFiles('./fixtures/avatar.jpg');

// Multiple files
await page.getByLabel('Upload documents').setInputFiles([
  './fixtures/invoice.pdf',
  './fixtures/receipt.pdf',
]);

// Clear file input
await page.getByLabel('Upload photo').setInputFiles([]);

// Handle file chooser dialog
const [fileChooser] = await Promise.all([
  page.waitForEvent('filechooser'),
  page.getByRole('button', { name: 'Choose file' }).click(),
]);
await fileChooser.setFiles('./fixtures/document.pdf');

// Upload file from buffer (no file on disk needed)
await page.getByLabel('Upload').setInputFiles({
  name: 'test.txt',
  mimeType: 'text/plain',
  buffer: Buffer.from('test file content'),
});
```

## Keyboard API

Low-level keyboard control:

```typescript
// Type a string (fires keydown, keypress, keyup for each character)
await page.keyboard.type('Hello, World!');

// Press and hold a key
await page.keyboard.down('Shift');
await page.keyboard.press('ArrowRight');
await page.keyboard.press('ArrowRight');
await page.keyboard.press('ArrowRight');
await page.keyboard.up('Shift');   // selects 3 characters to the right

// Insert text without key events (fastest, no keydown/keyup)
await page.keyboard.insertText('Pasted text');
```

## Mouse API

Low-level mouse control:

```typescript
// Move the cursor
await page.mouse.move(100, 200);

// Click at coordinates
await page.mouse.click(100, 200);
await page.mouse.click(100, 200, { button: 'right' });
await page.mouse.dblclick(100, 200);

// Press and release buttons
await page.mouse.down();
await page.mouse.move(300, 400, { steps: 20 }); // slow drag
await page.mouse.up();

// Scroll at position
await page.mouse.wheel(0, 300); // scroll 300px down
```

## page.evaluate — Run JavaScript in the Browser

Execute arbitrary JavaScript in the browser context:

```typescript
// Get a value from the page
const title = await page.evaluate(() => document.title);

// Manipulate the DOM
await page.evaluate(() => {
  document.querySelector('#overlay')?.remove();
});

// Pass arguments to the browser function
const itemText = await page.evaluate((selector) => {
  return document.querySelector(selector)?.textContent;
}, '.item:nth-child(3)');

// Return complex values
const state = await page.evaluate(() => ({
  scrollY: window.scrollY,
  innerWidth: window.innerWidth,
  userAgent: navigator.userAgent,
}));

// Execute on a locator element
const inputValue = await page.getByLabel('Email').evaluate(
  (el: HTMLInputElement) => el.value,
);

// Evaluate with ElementHandle
const button = await page.getByRole('button').elementHandle();
const isDisabled = await button?.evaluate((el: HTMLButtonElement) => el.disabled);
```

## page.exposeFunction — Expose Node.js Functions to the Browser

```typescript
// Make a Node.js function callable from browser scripts
await page.exposeFunction('readFile', async (path: string) => {
  const fs = await import('fs/promises');
  return fs.readFile(path, 'utf-8');
});

// Now the browser can call window.readFile()
const content = await page.evaluate(async () => {
  return await window.readFile('./data.json');
});
```

## scrollIntoViewIfNeeded

```typescript
// Scroll element into viewport if needed
await page.getByRole('button', { name: 'Load more' }).scrollIntoViewIfNeeded();

// Then click
await page.getByRole('button', { name: 'Load more' }).click();

// Scroll the page
await page.evaluate(() => window.scrollTo(0, document.body.scrollHeight));
await page.mouse.wheel(0, 500);  // scroll 500px down
```

## waitForLoadState

Wait for the page to reach a specific load state:

```typescript
// Wait for 'load' event (default after navigation)
await page.waitForLoadState('load');

// Wait for DOMContentLoaded (faster than 'load')
await page.waitForLoadState('domcontentloaded');

// Wait for network to be idle (no requests for 500ms)
// Use for SPAs that load data after initial page load
await page.waitForLoadState('networkidle');

// Combined navigation + load:
await Promise.all([
  page.waitForLoadState('networkidle'),
  page.getByRole('link', { name: 'Reports' }).click(),
]);
```

## waitForURL and waitForNavigation

```typescript
// Wait for URL to match
await page.waitForURL('/dashboard');
await page.waitForURL(/\/users\/\d+/);
await page.waitForURL('**/profile*');

// Wait for navigation to complete
await Promise.all([
  page.waitForNavigation({ url: '/success', waitUntil: 'networkidle' }),
  page.getByRole('button', { name: 'Submit' }).click(),
]);
```

:::caution[Avoid Arbitrary Waits]
Never use `page.waitForTimeout(5000)` as a substitute for proper event handling. It creates slow, flaky tests. Use `waitForLoadState`, `waitForURL`, `waitForResponse`, or `expect` assertions with retry instead.
:::

## Real-World Form Example

```typescript
test('creates a new product', async ({ page }) => {
  await page.goto('/admin/products/new');

  // Fill form fields
  await page.getByLabel('Product Name').fill('Wireless Headphones Pro');
  await page.getByLabel('SKU').fill('WHP-001');
  await page.getByLabel('Price').fill('149.99');
  await page.getByRole('combobox', { name: 'Category' }).selectOption('Electronics');
  await page.getByRole('combobox', { name: 'Status' }).selectOption('active');

  // Rich text editor (uses evaluate for complex interactions)
  const descriptionEditor = page.getByTestId('description-editor');
  await descriptionEditor.click();
  await descriptionEditor.type('High-quality wireless headphones with 30-hour battery life.');

  // Upload product image
  const [fileChooser] = await Promise.all([
    page.waitForEvent('filechooser'),
    page.getByRole('button', { name: 'Upload image' }).click(),
  ]);
  await fileChooser.setFiles('./fixtures/headphones.jpg');

  // Wait for upload to complete
  await expect(page.getByAltText('Product image preview')).toBeVisible();

  // Check the featured checkbox
  await page.getByRole('checkbox', { name: 'Featured product' }).check();

  // Submit
  await page.getByRole('button', { name: 'Create Product' }).click();

  // Verify success
  await expect(page).toHaveURL(/\/admin\/products\/\d+/);
  await expect(page.getByRole('alert')).toHaveText('Product created successfully');
  await expect(page.getByRole('heading', { name: 'Wireless Headphones Pro' })).toBeVisible();
});
```

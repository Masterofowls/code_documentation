CSS custom properties (often called "CSS variables") are one of the most powerful features in modern CSS. Unlike Sass variables that are compiled away, CSS custom properties are **live in the browser** — they participate in the cascade, can be updated with JavaScript, and enable dynamic theming.

## Declaring and Using Custom Properties

```css
/* Declare with -- prefix */
:root {
  --primary: #0066cc;
  --primary-hover: #004499;
  --radius: 0.5rem;
  --spacing-4: 1rem;
  --font-sans: 'Inter', system-ui, sans-serif;
}

/* Use with var() */
.button {
  background: var(--primary);
  border-radius: var(--radius);
  padding: var(--spacing-4);
  font-family: var(--font-sans);
}

.button:hover {
  background: var(--primary-hover);
}
```

## `var()` with Fallback Values

```css
.component {
  /* var(--property, fallback) */
  color: var(--text-color, #333);
  font-size: var(--font-size, 1rem);
  padding: var(--spacing, 1.5rem);

  /* Fallback can be another custom property */
  color: var(--brand-text, var(--text-primary, #1e293b));

  /* Fallback can be a complex value */
  background: var(--card-bg, linear-gradient(135deg, #fff, #f8fafc));

  /* Chain multiple fallbacks */
  font-family: var(--brand-font, var(--system-font, system-ui, sans-serif));
}
```

:::tip[Fallbacks for Robustness]
Fallback values make your styles robust against missing custom properties — useful when you're building a component library where consumers may not define all expected properties.
:::

## Scope: :root vs Local

Custom properties cascade just like regular CSS properties. A property defined on an element is available to all its descendants.

```css
/* Global scope — available everywhere */
:root {
  --color-primary: #0066cc;
  --color-background: #ffffff;
  --color-text: #1e293b;
  --spacing-base: 1rem;
}

/* Local scope — only within .card and its descendants */
.card {
  --card-padding: 1.5rem;
  --card-radius: 0.75rem;
  --card-shadow: 0 4px 6px -1px rgb(0 0 0 / 0.1);

  padding: var(--card-padding);
  border-radius: var(--card-radius);
  box-shadow: var(--card-shadow);
}

/* Override locally for a variant */
.card--featured {
  --card-padding: 2rem;
  --card-shadow: 0 10px 25px -5px rgb(0 0 0 / 0.2);
}

/* Component-level properties — closed namespace */
.tooltip {
  --tooltip-bg: #1e293b;
  --tooltip-text: #fff;
  --tooltip-padding: 0.375rem 0.75rem;
  --tooltip-radius: 0.375rem;

  background: var(--tooltip-bg);
  color: var(--tooltip-text);
  padding: var(--tooltip-padding);
  border-radius: var(--tooltip-radius);
}
```

## Updating Custom Properties with JavaScript

Custom properties updated via JavaScript trigger re-renders just like CSS class changes:

```js
const root = document.documentElement; // :root element

// Read a custom property
const primary = getComputedStyle(root).getPropertyValue('--color-primary').trim();
console.log(primary); // '#0066cc'

// Set a custom property
root.style.setProperty('--color-primary', '#8b5cf6');

// Remove a custom property (falls back to :root value or fallback)
root.style.removeProperty('--color-primary');

// Set on a specific element
const card = document.querySelector('.card');
card.style.setProperty('--card-padding', '2.5rem');

// Dynamic theming
function setTheme(theme) {
  if (theme === 'dark') {
    root.style.setProperty('--color-background', '#0f172a');
    root.style.setProperty('--color-text', '#f1f5f9');
    root.style.setProperty('--color-primary', '#60a5fa');
  } else {
    root.style.removeProperty('--color-background');
    root.style.removeProperty('--color-text');
    root.style.removeProperty('--color-primary');
  }
}
```

```js
// Theme toggle implementation
const STORAGE_KEY = 'preferred-theme';

function getPreferredTheme() {
  const stored = localStorage.getItem(STORAGE_KEY);
  if (stored) return stored;
  return window.matchMedia('(prefers-color-scheme: dark)').matches ? 'dark' : 'light';
}

function applyTheme(theme) {
  document.documentElement.setAttribute('data-theme', theme);
  localStorage.setItem(STORAGE_KEY, theme);

  const btn = document.getElementById('theme-toggle');
  if (btn) {
    btn.setAttribute('aria-label', `Switch to ${theme === 'dark' ? 'light' : 'dark'} mode`);
    btn.textContent = theme === 'dark' ? '☀️' : '🌙';
  }
}

// Apply on load
applyTheme(getPreferredTheme());

// Toggle on button click
document.getElementById('theme-toggle')?.addEventListener('click', () => {
  const current = document.documentElement.getAttribute('data-theme') ?? 'light';
  applyTheme(current === 'dark' ? 'light' : 'dark');
});
```

## `@property` — Typed Custom Properties

`@property` (CSS Houdini) defines custom properties with types and initial values, enabling animation of values that CSS normally can't interpolate:

```css
/* Register a typed custom property */
@property --rotation {
  syntax: '<angle>';     /* Valid CSS angle */
  inherits: false;
  initial-value: 0deg;
}

@property --progress {
  syntax: '<number>';
  inherits: false;
  initial-value: 0;
}

@property --hue {
  syntax: '<number>';
  inherits: false;
  initial-value: 230;
}

/* Now these custom properties can be animated! */
@keyframes spin-gradient {
  to { --rotation: 360deg; }
}

.gradient-spinner {
  background: conic-gradient(from var(--rotation), #0066cc, #8b5cf6, #0066cc);
  animation: spin-gradient 2s linear infinite;
  border-radius: 50%;
}

/* Animate a colour hue shift */
@keyframes hue-shift {
  to { --hue: 360; }
}

.rainbow {
  color: hsl(var(--hue) 80% 50%);
  animation: hue-shift 3s linear infinite;
}

/* Animate a progress property for a custom gauge */
@property --gauge-value {
  syntax: '<percentage>';
  inherits: false;
  initial-value: 0%;
}

.gauge {
  background: conic-gradient(
    #0066cc var(--gauge-value),
    #e2e8f0 var(--gauge-value)
  );
  transition: --gauge-value 0.6s cubic-bezier(0.34, 1.56, 0.64, 1);
}
```

## Design Token Patterns

### Semantic Token System

```css
/* Layer 1: Primitive/base tokens (raw values) */
:root {
  --blue-50:  #eff6ff;
  --blue-100: #dbeafe;
  --blue-200: #bfdbfe;
  --blue-500: #3b82f6;
  --blue-600: #2563eb;
  --blue-700: #1d4ed8;
  --blue-900: #1e3a8a;

  --neutral-50:  #f8fafc;
  --neutral-100: #f1f5f9;
  --neutral-200: #e2e8f0;
  --neutral-500: #64748b;
  --neutral-700: #334155;
  --neutral-900: #0f172a;
}

/* Layer 2: Semantic tokens (intent, not value) */
:root {
  --color-background:        var(--neutral-50);
  --color-surface:           #ffffff;
  --color-text-primary:      var(--neutral-900);
  --color-text-secondary:    var(--neutral-500);
  --color-border:            var(--neutral-200);
  --color-action-primary:    var(--blue-600);
  --color-action-primary-hover: var(--blue-700);
  --color-action-focus:      var(--blue-200);
}

/* Layer 3: Component tokens (scoped to component) */
.button--primary {
  --button-bg: var(--color-action-primary);
  --button-bg-hover: var(--color-action-primary-hover);
  --button-text: #ffffff;
  --button-focus-ring: var(--color-action-focus);

  background: var(--button-bg);
  color: var(--button-text);
}

.button--primary:hover {
  background: var(--button-bg-hover);
}
```

### Responsive Spacing Scale

```css
/* Fluid spacing that scales with viewport */
@property --space-1 { syntax: '<length>'; inherits: false; initial-value: 0.25rem; }
@property --space-4 { syntax: '<length>'; inherits: false; initial-value: 1rem; }

:root {
  /* Base scale */
  --space-1: 0.25rem;
  --space-2: 0.5rem;
  --space-3: 0.75rem;
  --space-4: 1rem;
  --space-6: 1.5rem;
  --space-8: 2rem;
  --space-12: 3rem;
  --space-16: 4rem;

  /* Fluid spacing using clamp */
  --space-section: clamp(3rem, 8vw, 6rem);
  --space-container: clamp(1rem, 4vw, 2rem);
}
```

## CSS Houdini

CSS Houdini is a set of low-level browser APIs that expose the CSS engine to JavaScript. `@property` is the most stable part of Houdini. Other features include:

- **Paint API** (`CSS.paintWorklet`) — custom `background-image` rendering
- **Layout API** — define custom layout algorithms
- **Typed OM** (`element.attributeStyleMap`) — type-safe CSS value manipulation

```js
// CSS Paint API example
if ('paintWorklet' in CSS) {
  CSS.paintWorklet.addModule('/worklets/confetti-painter.js');
}
```

```css
/* Use the custom painter */
.confetti-button {
  background-image: paint(confetti);
}
```

:::tip[Houdini Browser Support]
`@property` has excellent browser support (Chrome 85+, Firefox 128+, Safari 16.4+). The Paint API, Layout API, and Typed OM have more limited support — check [ishoudinireadyyet.com](https://ishoudinireadyyet.com/) for current status.
:::

CSS layout has been transformed by Flexbox and Grid. Understanding all the layout tools — and when to use each — gives you complete control over any design.

## Normal Flow

By default, elements follow **normal flow**:

- **Block elements** (`div`, `p`, `h1`) stack vertically, each on its own line, taking up full available width
- **Inline elements** (`span`, `a`, `strong`) flow horizontally with text, wrapping when they reach the container edge

```css
/* Force block or inline behaviour */
span {
  display: block;
} /* Now stacks vertically */
div {
  display: inline;
} /* Now flows with text */
span {
  display: inline-block;
} /* Inline flow, but accepts width/height */
```

## Flexbox

Flexbox is a **one-dimensional** layout model — it arranges items along a single axis (row or column).

### Container Properties

```css
.flex-container {
  display: flex; /* or inline-flex */

  /* Direction of the main axis */
  flex-direction: row; /* → left to right (default) */
  flex-direction: row-reverse; /* ← right to left */
  flex-direction: column; /* ↓ top to bottom */
  flex-direction: column-reverse; /* ↑ bottom to top */

  /* Wrapping */
  flex-wrap: nowrap; /* Single line (default) */
  flex-wrap: wrap; /* Wrap onto multiple lines */
  flex-wrap: wrap-reverse; /* Wrap, new lines above */

  /* Shorthand */
  flex-flow: row wrap;

  /* Main axis alignment (justify-content) */
  justify-content: flex-start; /* Pack items at start */
  justify-content: flex-end; /* Pack items at end */
  justify-content: center; /* Centre items */
  justify-content: space-between; /* Even gaps, no outer margin */
  justify-content: space-around; /* Even gaps, half outer margin */
  justify-content: space-evenly; /* Equal gaps everywhere */

  /* Cross axis alignment (align-items) */
  align-items: stretch; /* Fill cross axis (default) */
  align-items: flex-start; /* Align to start */
  align-items: flex-end; /* Align to end */
  align-items: center; /* Centre on cross axis */
  align-items: baseline; /* Align text baselines */

  /* Multi-line cross axis (when flex-wrap: wrap) */
  align-content: flex-start;
  align-content: center;
  align-content: space-between;
  align-content: space-around;

  /* Gap between flex items */
  gap: 1rem;
  gap: 1rem 2rem; /* row-gap column-gap */
  row-gap: 1rem;
  column-gap: 2rem;
}
```

### Item Properties

```css
.flex-item {
  /* Growth: how much the item grows relative to others */
  flex-grow: 0; /* Don't grow (default) */
  flex-grow: 1; /* Take up available space equally with other flex-grow: 1 items */
  flex-grow: 2; /* Takes twice as much space as flex-grow: 1 items */

  /* Shrink: how much the item shrinks when space is tight */
  flex-shrink: 1; /* Shrink equally (default) */
  flex-shrink: 0; /* Don't shrink — useful for fixed-size items */

  /* Basis: the item's initial size before growing/shrinking */
  flex-basis: auto; /* Use the item's content size (default) */
  flex-basis: 0; /* Start from zero — combined with flex-grow */
  flex-basis: 200px; /* Fixed starting size */
  flex-basis: 33.333%; /* Percentage of container */

  /* Shorthand: flex-grow flex-shrink flex-basis */
  flex: 0 1 auto; /* default */
  flex: 1; /* flex-grow: 1, flex-shrink: 1, flex-basis: 0 */
  flex: 1 1 0; /* equal distribution from zero basis */
  flex: 0 0 200px; /* fixed 200px, no grow or shrink */
  flex: auto; /* flex: 1 1 auto */
  flex: none; /* flex: 0 0 auto */

  /* Override container alignment for this item */
  align-self: auto; /* Use container's align-items */
  align-self: flex-start;
  align-self: flex-end;
  align-self: center;
  align-self: stretch;

  /* Visual order (doesn't change DOM order) */
  order: 0; /* Default */
  order: -1; /* Move to the front */
  order: 999; /* Move to the end */
}
```

### Common Flexbox Patterns

```css
/* Perfect centering */
.centered {
  display: flex;
  align-items: center;
  justify-content: center;
}

/* Sticky footer */
body {
  display: flex;
  flex-direction: column;
  min-height: 100vh;
}

main {
  flex: 1;
} /* Takes all available space, pushing footer down */

/* Holy Grail layout */
.page {
  display: flex;
  flex-direction: column;
}

.main-area {
  display: flex;
  flex: 1;
}

.sidebar {
  flex: 0 0 240px;
}
.content {
  flex: 1;
}
.aside {
  flex: 0 0 200px;
}

/* Navigation with logo left, links right */
.nav {
  display: flex;
  align-items: center;
  justify-content: space-between;
}
```

## CSS Grid

CSS Grid is a **two-dimensional** layout model — it places items into rows _and_ columns simultaneously.

### Container Properties

```css
.grid-container {
  display: grid;

  /* Define columns */
  grid-template-columns: 200px 1fr 200px; /* Fixed, flexible, fixed */
  grid-template-columns: repeat(3, 1fr); /* 3 equal columns */
  grid-template-columns: repeat(12, 1fr); /* 12-column grid */
  grid-template-columns: minmax(200px, 1fr) 3fr; /* min 200px, max 1fr */

  /* Define rows */
  grid-template-rows: auto 1fr auto; /* Header, content, footer */
  grid-template-rows: repeat(3, minmax(100px, auto));

  /* Named areas */
  grid-template-areas:
    'header  header  header'
    'sidebar content aside'
    'footer  footer  footer';

  /* Gaps */
  gap: 1.5rem;
  row-gap: 2rem;
  column-gap: 1rem;

  /* Alignment */
  justify-items: stretch; /* Align items horizontally in their cell */
  align-items: stretch; /* Align items vertically in their cell */
  justify-content: start; /* Align tracks in the container */
  align-content: start;

  /* Implicit grid: auto-create rows/columns for overflow items */
  grid-auto-rows: minmax(100px, auto);
  grid-auto-columns: 1fr;
  grid-auto-flow: row; /* Fill rows first (default) */
  grid-auto-flow: column;
  grid-auto-flow: row dense; /* Fill gaps greedily */
}
```

### Item Placement

```css
.grid-item {
  /* Place by line numbers */
  grid-column: 1 / 3; /* Start at line 1, end at line 3 */
  grid-column: 1 / span 2; /* Start at 1, span 2 columns */
  grid-column: span 2; /* Span 2, auto-place */
  grid-row: 1 / 3;
  grid-row: 1 / span 2;

  /* Place by area name */
  grid-area: header;
  grid-area: sidebar;

  /* Alignment override */
  justify-self: center;
  align-self: end;
}
```

### `auto-fill` vs `auto-fit`

```css
/* auto-fill: create as many columns as will fit — even if some are empty */
.auto-fill-grid {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));
  /* At 600px: 3 columns of 200px (extra empty columns may exist) */
}

/* auto-fit: collapse empty tracks — items fill available space */
.auto-fit-grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
  /* At 600px: 3 columns. At 300px: 1 column that stretches to fill */
}
```

### Subgrid

Subgrid lets child items participate in the parent grid's track sizing:

```css
.card-grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 1.5rem;
}

/* Each card is a subgrid participant */
.card {
  display: grid;
  grid-template-rows: auto 1fr auto; /* title, body, footer */
  grid-row: span 3;
  /* Inherit column tracks from parent */
  grid-column: subgrid; /* cards align their columns to the parent grid */
}

/* Card rows also use subgrid — every card's title/body/footer aligns */
.card {
  display: grid;
  grid-row: span 3;
  grid-template-rows: subgrid;
}
```

## Positioning

```css
/* static — normal flow (default) */
.normal {
  position: static;
}

/* relative — offset from normal position; creates a containing block */
.relative-box {
  position: relative;
  top: 10px; /* Move down 10px from where it would naturally be */
  left: -5px; /* Move left 5px */
  /* Still takes up its original space in the flow */
}

/* absolute — removed from flow, positioned relative to nearest positioned ancestor */
.absolute-box {
  position: absolute;
  top: 0;
  right: 0;
  /* Positioned relative to nearest ancestor with position !== static */
}

/* fixed — like absolute but positioned relative to the viewport */
.fixed-nav {
  position: fixed;
  top: 0;
  left: 0;
  right: 0;
  z-index: 100;
  /* Stays in place as page scrolls */
}

/* sticky — switches between relative and fixed based on scroll position */
.sticky-header {
  position: sticky;
  top: 0; /* Sticks 0px from top of scroll container */
  z-index: 10;
  /* Acts like relative until scroll brings it to top, then acts like fixed */
}
```

## Z-Index and Stacking Contexts

`z-index` only works on positioned elements (`relative`, `absolute`, `fixed`, `sticky`) and flex/grid items. A **stacking context** is an isolated layer — elements inside can only z-stack relative to each other.

```css
.stacking-context {
  /* Any of these create a new stacking context: */
  position: relative;
  z-index: 1;
  position: absolute;
  z-index: 1;
  opacity: 0.99; /* less than 1 */
  transform: translateZ(0);
  will-change: transform;
  isolation: isolate; /* explicit stacking context */
  filter: blur(0);
  contain: layout;
}

/* isolation: isolate is the cleanest way to create a stacking context */
.modal-root {
  isolation: isolate; /* All modals z-index relative to this, not the whole page */
}

.modal {
  z-index: 1;
} /* Inside .modal-root */
.toast {
  z-index: 100;
} /* Also inside .modal-root — OK, they're siblings */
```

## Multi-Column Layout

```css
.article-text {
  /* Fixed number of columns */
  column-count: 3;

  /* Or minimum column width (browser decides count) */
  column-width: 200px;

  /* Gap between columns */
  column-gap: 2rem;

  /* Decorative line between columns */
  column-rule: 1px solid #e2e8f0;
  column-rule: 2px dashed #cbd5e1;

  /* Span an element across all columns */
  .pull-quote {
    column-span: all;
  }

  /* Control column breaks */
  h2,
  h3 {
    break-after: avoid;
  } /* Don't break right after headings */
  figure {
    break-inside: avoid;
  } /* Don't break inside figures */
}
```

## Float (Legacy Context)

Floats are largely replaced by Flexbox and Grid for layout. However, they still serve a purpose for text wrapping around images:

```css
/* Float image with text wrap */
figure.float-left {
  float: left;
  margin: 0 1.5rem 1rem 0;
  max-width: 40%;
}

figure.float-right {
  float: right;
  margin: 0 0 1rem 1.5rem;
  max-width: 40%;
}

/* Clear floats in the parent (modern clearfix) */
.clearfix::after {
  content: '';
  display: table;
  clear: both;
}

/* Even simpler: overflow creates a new BFC that contains floats */
.container {
  overflow: hidden; /* or auto */
}

/* Best modern option: display: flow-root */
.container {
  display: flow-root; /* Creates BFC without side effects */
}
```

:::tip[Choose the Right Layout Tool]

- **Flexbox**: one-dimensional layouts, nav bars, card rows, centering content
- **Grid**: two-dimensional layouts, full-page layouts, complex component grids
- **Position relative/absolute**: tooltips, badges, overlays on top of existing layout
- **Position fixed/sticky**: fixed headers, sticky sidebars, floating CTAs
- **Multi-column**: newspaper-style text columns, masonry-like lists
  :::

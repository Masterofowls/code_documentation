CSS layout defines how elements are sized and positioned relative to each other. Modern CSS gives you powerful, purpose-built tools for every layout challenge.

## Normal Flow

Without any CSS layout overrides, elements follow **normal flow**:

- **Block elements** stack vertically, each on a new line, taking full available width
- **Inline elements** flow horizontally like words in text, wrapping to new lines

```css
/* Block formatting context — default for block elements */
div, p, h1, section { display: block; }

/* Inline formatting context — default for inline elements */
span, a, strong, em { display: inline; }

/* Inline-block: inline flow but can have width/height/padding/margin */
.badge { display: inline-block; padding: 0.25em 0.5em; }
```

## Flexbox

Flexbox is a one-dimensional layout method — it lays items out in a row **or** a column.

### Flex Container Properties

```css
.container {
  display: flex; /* or inline-flex */

  /* ─── Direction ─────────────────────────────────────── */
  flex-direction: row;            /* Default: left to right */
  flex-direction: row-reverse;    /* Right to left */
  flex-direction: column;         /* Top to bottom */
  flex-direction: column-reverse; /* Bottom to top */

  /* ─── Wrapping ───────────────────────────────────────── */
  flex-wrap: nowrap;  /* Default: single line (items may shrink) */
  flex-wrap: wrap;    /* Items wrap to next line */
  flex-wrap: wrap-reverse;

  /* Shorthand for direction + wrap */
  flex-flow: row wrap;

  /* ─── Alignment on main axis (row = horizontal) ───────── */
  justify-content: flex-start;    /* Default: items at start */
  justify-content: flex-end;
  justify-content: center;
  justify-content: space-between; /* Gap between items, none at edges */
  justify-content: space-around;  /* Equal space around each item */
  justify-content: space-evenly;  /* Equal space between items and edges */

  /* ─── Alignment on cross axis (row = vertical) ─────────── */
  align-items: stretch;     /* Default: stretch to fill container height */
  align-items: flex-start;
  align-items: flex-end;
  align-items: center;
  align-items: baseline;    /* Align text baselines */

  /* ─── Multi-line alignment (when flex-wrap: wrap) ────── */
  align-content: stretch;
  align-content: flex-start;
  align-content: center;
  align-content: space-between;

  /* ─── Gap ────────────────────────────────────────────── */
  gap: 1rem;           /* row-gap and column-gap equally */
  gap: 1rem 2rem;      /* row-gap column-gap */
  row-gap: 1rem;
  column-gap: 2rem;
}
```

### Flex Item Properties

```css
.item {
  /* flex-grow: how much free space the item takes relative to siblings */
  flex-grow: 0;  /* Default: don't grow */
  flex-grow: 1;  /* Take available space equally with other flex-grow: 1 items */
  flex-grow: 2;  /* Take twice as much space as flex-grow: 1 siblings */

  /* flex-shrink: how much the item shrinks when container is too small */
  flex-shrink: 1; /* Default: shrink equally */
  flex-shrink: 0; /* Don't shrink at all */

  /* flex-basis: initial size before free space is distributed */
  flex-basis: auto;    /* Use width/height property */
  flex-basis: 200px;   /* Start at 200px, then grow/shrink */
  flex-basis: 0;       /* Ignore natural size — divide space purely by flex-grow */

  /* flex: shorthand (grow shrink basis) */
  flex: 0 1 auto;      /* Default */
  flex: 1;             /* Shorthand for flex: 1 1 0 */
  flex: auto;          /* Shorthand for flex: 1 1 auto */
  flex: none;          /* Shorthand for flex: 0 0 auto (rigid) */

  /* align-self: override the container's align-items for this item */
  align-self: auto;      /* Inherit from container */
  align-self: flex-start;
  align-self: center;
  align-self: flex-end;
  align-self: stretch;

  /* order: change visual order without changing DOM order */
  /* Accessible order follows DOM, visual follows order */
  order: 0;    /* Default */
  order: -1;   /* Before all order: 0 items */
  order: 1;    /* After all order: 0 items */
}
```

### Common Flexbox Patterns

```css
/* Horizontal centring */
.centered {
  display: flex;
  justify-content: center;
  align-items: center;
}

/* Navigation bar */
.navbar {
  display: flex;
  align-items: center;
  gap: 1rem;
}

.navbar .spacer { flex: 1; } /* Pushes right-side items to the end */

/* Card grid with equal-height cards */
.card-grid {
  display: flex;
  flex-wrap: wrap;
  gap: 1.5rem;
}

.card {
  flex: 1 1 300px; /* Grow, shrink, min 300px basis */
  display: flex;
  flex-direction: column;
}

.card-body { flex: 1; } /* Push footer to bottom of card */
```

## CSS Grid

CSS Grid is a two-dimensional layout system — rows **and** columns simultaneously.

### Grid Container Properties

```css
.grid {
  display: grid; /* or inline-grid */

  /* ─── Define columns ─────────────────────────────────── */
  grid-template-columns: 200px 1fr 1fr;        /* Fixed + fluid */
  grid-template-columns: repeat(3, 1fr);        /* 3 equal columns */
  grid-template-columns: repeat(auto-fill, minmax(250px, 1fr)); /* Responsive! */
  grid-template-columns: repeat(auto-fit,  minmax(250px, 1fr)); /* Collapses empty tracks */

  /* ─── Define rows ─────────────────────────────────────── */
  grid-template-rows: auto 1fr auto; /* header, main, footer */
  grid-template-rows: repeat(3, 100px);

  /* ─── Named areas ─────────────────────────────────────── */
  grid-template-areas:
    "header  header  header"
    "sidebar main    main  "
    "footer  footer  footer";

  /* ─── Shorthand ───────────────────────────────────────── */
  grid-template:
    "header"  60px
    "main"    1fr
    "footer"  60px
    / 1fr;     /* column tracks */

  /* ─── Gap ─────────────────────────────────────────────── */
  gap: 1rem;
  row-gap: 1rem;
  column-gap: 1.5rem;

  /* ─── Implicit tracks (auto-created) ─────────────────── */
  grid-auto-rows: minmax(100px, auto); /* Height of implicitly added rows */
  grid-auto-columns: 200px;
  grid-auto-flow: row;    /* Default: fill rows first */
  grid-auto-flow: column; /* Fill columns first */
  grid-auto-flow: row dense; /* Fill holes with later items */

  /* ─── Alignment ───────────────────────────────────────── */
  justify-items: stretch;   /* Align items horizontally in their cell */
  align-items: stretch;     /* Align items vertically in their cell */
  place-items: center;      /* Shorthand for align-items + justify-items */

  justify-content: start;   /* Align the whole grid in the container */
  align-content: start;
  place-content: center;
}
```

### Grid Item Properties

```css
.item {
  /* ─── Placement by line number ───────────────────────── */
  grid-column: 1 / 3;       /* Start at line 1, end at line 3 */
  grid-column: 1 / span 2;  /* Start at line 1, span 2 columns */
  grid-column: 2 / -1;      /* From line 2 to last line */
  grid-column: span 2;      /* Span 2, browser decides start */

  grid-row: 1 / 2;
  grid-row: span 2;

  /* ─── Placement by area name ─────────────────────────── */
  grid-area: header;
  grid-area: sidebar;
  grid-area: main;
  grid-area: footer;

  /* ─── Alignment within cell ──────────────────────────── */
  justify-self: center;  /* Horizontal */
  align-self: end;       /* Vertical */
  place-self: center;    /* Shorthand */
}
```

### Grid Layout Examples

```css
/* ─── Holy Grail Layout ────────────────────────────────── */
.page {
  display: grid;
  grid-template-areas:
    "header  header  header"
    "sidebar main    aside "
    "footer  footer  footer";
  grid-template-columns: 250px 1fr 250px;
  grid-template-rows: auto 1fr auto;
  min-height: 100dvh;
}

header  { grid-area: header;  }
.sidebar{ grid-area: sidebar; }
main    { grid-area: main;    }
aside   { grid-area: aside;   }
footer  { grid-area: footer;  }

/* ─── Responsive card grid (no media queries!) ──────────── */
.cards {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(min(100%, 280px), 1fr));
  gap: 1.5rem;
}
```

### `auto-fill` vs `auto-fit`

```css
/* auto-fill: creates as many tracks as possible, even empty ones */
/* Result: if 3 items fit, creates 5 columns — 2 are empty */
.fill { grid-template-columns: repeat(auto-fill, minmax(200px, 1fr)); }

/* auto-fit: collapses empty tracks — items grow to fill space */
/* Result: if 3 items fit perfectly, no empty columns */
.fit  { grid-template-columns: repeat(auto-fit,  minmax(200px, 1fr)); }
```

### Subgrid

Align nested elements to the parent grid:

```css
.parent-grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 1rem;
}

.card {
  grid-column: span 1;
  display: grid;
  /* Inherit parent's column tracks */
  grid-template-columns: subgrid;
  /* Inherit parent's row tracks */
  grid-template-rows: subgrid;
  grid-row: span 3;
}
```

## Position

```css
.element {
  /* static: default — follows normal flow */
  position: static;

  /* relative: offset from normal position, still in flow */
  position: relative;
  top: 10px;    /* Move down 10px from normal position */
  left: -5px;   /* Move left 5px */
  /* Does NOT affect surrounding elements */

  /* absolute: removed from flow, positioned relative to nearest positioned ancestor */
  position: absolute;
  top: 0;
  right: 0;
  /* If no positioned ancestor: relative to initial containing block (<html>) */

  /* fixed: relative to viewport — stays visible during scroll */
  position: fixed;
  bottom: 1rem;
  right: 1rem;

  /* sticky: hybrid — relative until scroll threshold, then fixed */
  position: sticky;
  top: 0; /* Sticks to top of viewport when scrolled past */
  /* Must have an overflow-visible ancestor to work */
}
```

## Z-index and Stacking Context

```css
/* z-index only works on positioned elements (not static) */
.modal-overlay  { position: fixed; z-index: 100; }
.modal          { position: fixed; z-index: 101; }
.tooltip        { position: absolute; z-index: 50; }

/* Creating a stacking context limits z-index to within the element */
/* These all create a new stacking context: */
.stacking-context {
  opacity: 0.99;              /* Any opacity < 1 */
  transform: translateZ(0);   /* Any transform */
  filter: blur(0);            /* Any filter */
  will-change: transform;
  isolation: isolate;         /* Most explicit and predictable */
  position: relative; z-index: 0;
}
```

:::tip[Use `isolation: isolate` to Contain Z-index]
When a component (like a card, modal, or dropdown) needs its own z-index space without interfering with the page's z-index, add `isolation: isolate` to its container. This is the cleanest, most explicit way to create a stacking context.
:::

## Multi-column Layout

```css
/* Define columns by count */
.newspaper {
  column-count: 3;
  column-gap: 2rem;
  column-rule: 1px solid #e5e7eb; /* Line between columns */
}

/* Define columns by width (browser determines count) */
.auto-columns {
  column-width: 300px;
  column-gap: 1.5rem;
}

/* Make an element span all columns */
.pull-quote {
  column-span: all;
}

/* Control column breaks */
h2 { break-after: avoid; }  /* Don't break after headings */
.no-break { break-inside: avoid; } /* Keep element on one page/column */
```

## Float (Legacy)

Float is a legacy layout tool, largely replaced by Flexbox and Grid. It still has valid uses for text wrap around images:

```css
/* Float an image with text wrap */
.article img {
  float: left;
  margin: 0 1.5rem 1rem 0;
  border-radius: 4px;
}

/* Clearfix — modern approach */
.clearfix::after {
  content: '';
  display: table;
  clear: both;
}

/* Or use display: flow-root on the container */
.container {
  display: flow-root; /* Creates BFC, clears floats */
}
```

:::caution[Don't Use Float for Layout]
Using float for multi-column page layout creates fragile, hard-to-maintain code. Use Flexbox for one-dimensional layouts and CSS Grid for two-dimensional layouts. Reserve float only for the text-wrapping-around-an-image use case.
:::

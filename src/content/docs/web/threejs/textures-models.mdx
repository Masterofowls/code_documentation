Textures and 3D models are the assets that bring Three.js scenes to life. This guide covers the full lifecycle — from loading a PNG with `TextureLoader` to importing a compressed GLTF model with Draco geometry and KTX2 textures.

---

## TextureLoader

The simplest way to load a texture from a URL:

```js
import * as THREE from 'three';

const loader = new THREE.TextureLoader();

// Basic load
const texture = loader.load('/textures/wood_diffuse.jpg');

// With callbacks
const texture2 = loader.load(
  '/textures/metal_normal.jpg',
  (tex) => {
    console.log('Loaded:', tex.image.width, '×', tex.image.height);
  },
  (event) => {
    console.log('Progress:', event.loaded / event.total * 100, '%');
  },
  (error) => {
    console.error('Load error:', error);
  }
);

// Apply to a material
const material = new THREE.MeshStandardMaterial({
  map:       loader.load('/textures/albedo.jpg'),
  normalMap: loader.load('/textures/normal.jpg'),
  roughnessMap: loader.load('/textures/roughness.jpg'),
  metalnessMap: loader.load('/textures/metalness.jpg'),
  aoMap:     loader.load('/textures/ao.jpg'),
});
```

## LoadingManager

Coordinate multiple async loads and show a loading screen:

```js
const manager = new THREE.LoadingManager(
  // onLoad — all assets complete
  () => {
    document.querySelector('#loading-screen').style.display = 'none';
    scene.add(model);
  },
  // onProgress — per-item progress
  (url, loaded, total) => {
    const pct = Math.round(loaded / total * 100);
    document.querySelector('#progress').textContent = `${pct}%`;
    console.log(`[${loaded}/${total}] ${url}`);
  },
  // onError
  (url) => {
    console.error('Failed to load:', url);
  }
);

const texLoader  = new THREE.TextureLoader(manager);
const gltfLoader = new GLTFLoader(manager);

const albedo = texLoader.load('/textures/albedo.jpg');
gltfLoader.load('/models/scene.glb', (gltf) => {
  model = gltf.scene;
});
// Manager fires onLoad when both complete
```

## Texture Properties

After loading, configure how the texture is applied:

```js
const texture = new THREE.TextureLoader().load('/textures/tiles.jpg');

// Wrapping — what to do when UV coordinates go outside [0,1]
texture.wrapS = THREE.RepeatWrapping;      // horizontal
texture.wrapT = THREE.RepeatWrapping;      // vertical
// THREE.ClampToEdgeWrapping  — stretch edge pixels
// THREE.MirroredRepeatWrapping — tile with mirroring

// Repeat (tile the texture)
texture.repeat.set(4, 4);    // tile 4× in each direction

// Offset (scroll the texture)
texture.offset.set(0.5, 0);  // shift half a tile in U

// Rotation (around the pivot)
texture.rotation = Math.PI / 4;      // 45°
texture.center.set(0.5, 0.5);        // rotate around centre

// Filtering — how to sample between pixels
texture.magFilter = THREE.LinearFilter;         // up-scaling (default)
texture.minFilter = THREE.LinearMipmapLinearFilter; // down-scaling (default)
// For pixel art / non-blurry textures:
texture.magFilter = THREE.NearestFilter;
texture.minFilter = THREE.NearestFilter;
texture.generateMipmaps = false;               // disable for NearestFilter

// Encoding — important for colour textures (not normal/roughness maps!)
texture.colorSpace = THREE.SRGBColorSpace;     // albedo / emissive maps
// Normal/roughness/metalness maps keep LinearSRGBColorSpace (default)

// Anisotropy — sharpens at oblique viewing angles
const maxAniso = renderer.capabilities.getMaxAnisotropy();
texture.anisotropy = maxAniso;                 // 8 or 16 is typical max
```

## Special Texture Types

### CubeTextureLoader (Skybox)

```js
const cubeLoader = new THREE.CubeTextureLoader();
const skybox = cubeLoader.load([
  '/textures/sky/px.jpg', // +X (right)
  '/textures/sky/nx.jpg', // -X (left)
  '/textures/sky/py.jpg', // +Y (top)
  '/textures/sky/ny.jpg', // -Y (bottom)
  '/textures/sky/pz.jpg', // +Z (front)
  '/textures/sky/nz.jpg', // -Z (back)
]);
scene.background = skybox;
scene.environment = skybox; // use as IBL
```

### VideoTexture

```js
const video = document.createElement('video');
video.src = '/videos/screen_content.mp4';
video.loop = true;
video.muted = true;
video.play();

const videoTex = new THREE.VideoTexture(video);
videoTex.colorSpace = THREE.SRGBColorSpace;

const screenMaterial = new THREE.MeshBasicMaterial({ map: videoTex });
// VideoTexture updates automatically each frame from the <video> element
```

### CanvasTexture

```js
const canvas = document.createElement('canvas');
canvas.width = 512;
canvas.height = 512;
const ctx = canvas.getContext('2d');

ctx.fillStyle = '#1e1b4b';
ctx.fillRect(0, 0, 512, 512);
ctx.fillStyle = '#ffffff';
ctx.font = 'bold 64px sans-serif';
ctx.textAlign = 'center';
ctx.fillText('Three.js', 256, 280);

const canvasTex = new THREE.CanvasTexture(canvas);

// Update texture after canvas changes
ctx.clearRect(0, 0, 512, 512);
ctx.fillText('Updated!', 256, 280);
canvasTex.needsUpdate = true;  // ← triggers GPU re-upload
```

### DataTexture

Generate procedural textures entirely in JavaScript:

```js
const size   = 256;
const data   = new Uint8Array(4 * size * size);

for (let i = 0; i < size * size; i++) {
  const x = i % size;
  const y = Math.floor(i / size);
  // Checkerboard
  const isLight = (Math.floor(x / 16) + Math.floor(y / 16)) % 2 === 0;
  const value = isLight ? 255 : 64;
  data[i * 4 + 0] = value;  // R
  data[i * 4 + 1] = value;  // G
  data[i * 4 + 2] = value;  // B
  data[i * 4 + 3] = 255;   // A
}

const dataTex = new THREE.DataTexture(data, size, size, THREE.RGBAFormat);
dataTex.needsUpdate = true;
```

## Loading GLTF Models

GLTF 2.0 (`.gltf` / `.glb`) is the preferred format for Three.js. `.glb` is the binary single-file variant.

```js
import { GLTFLoader } from 'three/addons/loaders/GLTFLoader.js';

const loader = new GLTFLoader();

loader.load('/models/helmet.glb', (gltf) => {
  console.log('GLTF loaded:', gltf);

  // gltf.scene      — the root THREE.Group
  // gltf.scenes     — array of scenes
  // gltf.animations — AnimationClip[]
  // gltf.cameras    — camera definitions from the file
  // gltf.asset      — metadata { version, generator }

  const model = gltf.scene;

  // Center the model
  const box = new THREE.Box3().setFromObject(model);
  const center = box.getCenter(new THREE.Vector3());
  model.position.sub(center);

  // Scale to fit in a 2-unit sphere
  const size = box.getSize(new THREE.Vector3()).length();
  model.scale.multiplyScalar(2 / size);

  scene.add(model);

  // Play all animations
  const mixer = new THREE.AnimationMixer(model);
  gltf.animations.forEach(clip => mixer.clipAction(clip).play());
}, undefined, (error) => {
  console.error('GLTF load error:', error);
});
```

## Draco Compression

Draco can reduce GLTF geometry size by 70–95%:

```bash
npm install three
# DRACOLoader is included in three/addons
```

```js
import { GLTFLoader } from 'three/addons/loaders/GLTFLoader.js';
import { DRACOLoader } from 'three/addons/loaders/DRACOLoader.js';

const dracoLoader = new DRACOLoader();
// Point to the Draco decoder — copy from node_modules/three/examples/jsm/libs/draco/
dracoLoader.setDecoderPath('/draco/');
dracoLoader.setDecoderConfig({ type: 'js' }); // or 'wasm' for better performance

const loader = new GLTFLoader();
loader.setDRACOLoader(dracoLoader);

loader.load('/models/compressed.glb', (gltf) => {
  scene.add(gltf.scene);
});
```

In your `vite.config.js`, copy the decoder:

```js
import { viteStaticCopy } from 'vite-plugin-static-copy';

export default {
  plugins: [
    viteStaticCopy({
      targets: [{ src: 'node_modules/three/examples/jsm/libs/draco', dest: '' }],
    }),
  ],
};
```

## KTX2 Compressed Textures

KTX2 with Basis Universal compression drastically reduces GPU memory and enables hardware-accelerated texture decompression:

```bash
npm install three
```

```js
import { KTX2Loader } from 'three/addons/loaders/KTX2Loader.js';
import { MeshoptDecoder } from 'three/addons/libs/meshopt_decoder.module.js';

const ktx2Loader = new KTX2Loader();
ktx2Loader.setTranscoderPath('/basis/');  // copy from three/examples/jsm/libs/basis/
ktx2Loader.detectSupport(renderer);

// Pair with GLTFLoader for models that reference .ktx2 textures
loader.setKTX2Loader(ktx2Loader);
loader.setMeshoptDecoder(MeshoptDecoder);
loader.load('/models/optimised.glb', (gltf) => {
  scene.add(gltf.scene);
});
```

GLTF 2.0 is the recommended format. It is smaller, supports PBR materials, animations, morph targets, and has first-class Three.js support. Use Blender's built-in GLTF exporter to convert older formats.
:::

## Model Optimisation

| Technique | Savings | Tool |
|---|---|---|
| Draco geometry compression | 70–95% mesh size | Blender, gltf-pipeline |
| KTX2 / Basis texture compression | 6–8× GPU memory | toktx, gltf-transform |
| Texture atlasing | Fewer draw calls | Blender, Materialize |
| Mesh LOD (Level of Detail) | Performance on distance | `THREE.LOD` class |
| Polygon reduction | Faster rendering | Blender Decimate, Simplygon |
| Remove hidden geometry | Smaller file | Blender |
| Merge meshes | Fewer draw calls | Blender, gltf-transform |

### Using THREE.LOD

```js
const lod = new THREE.LOD();

// High poly close-up model (< 5 units away)
const highDetail = new THREE.Mesh(
  new THREE.SphereGeometry(1, 64, 64),
  material,
);
lod.addLevel(highDetail, 0);

// Medium poly (5–20 units)
const medDetail = new THREE.Mesh(
  new THREE.SphereGeometry(1, 16, 16),
  material,
);
lod.addLevel(medDetail, 5);

// Low poly (> 20 units)
const lowDetail = new THREE.Mesh(
  new THREE.SphereGeometry(1, 6, 6),
  material,
);
lod.addLevel(lowDetail, 20);

scene.add(lod);
// Three.js automatically picks the right LOD level each frame
```
